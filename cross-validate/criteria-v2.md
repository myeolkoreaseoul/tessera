# 비목별 판정 기준 v2 (4모델 교차검증용)

> 대구경북첨단의료산업진흥재단 - 디지털헬스케어 의료기기 실증지원 사업
> 이 기관은 사업의 전문기관(수행기관)입니다.

## 공통 규칙

1. **0원 항목** → SKIP (취소전표)
2. **evidenceSub**에 "소득 지급명세서" 포함 → 시스템 내장 증빙 존재 (파일 없어도 증빙 인정)
3. **evidenceType**이 "보조금전용카드" → 카드전표로 세금계산서 대체 가능
4. **evidenceType**이 "전자세금계산서" 또는 "전자계산서" → 시스템 내장 결제증빙
5. **files=0개 + evidenceSub=""** → 증빙 없음 → **확인**
6. **보수적 원칙**: 확실한 경우만 "적정", 불확실하면 "확인"
7. "기타(첨부파일필수확인)" → 별도 첨부파일이 필요하다는 의미 → files 확인 필수

## ★ 오판 방지 규칙 (필독!)

### OP-1: "집행취소 협조 요청" 파일 ≠ 취소
- "2025년 디지털헬스케어 의료기기 실증지원 사업 집행취소 협조 요청(e나라도움).pdf"는 **단순 참고문서**
- 금액이 0원이 아닌 한 해당 항목은 유효한 집행임
- 이 파일이 있다고 "취소건"으로 판정하면 안 됨

### OP-2: 출장복명서 = 선택 증빙
- 출장복명서(출장보고서)는 **필수 증빙이 아님**
- 필수는: 출장신청서(근무지외출장/근무지내출장) + 지출결의서 + 교통비 영수증
- 출장복명서 미첨부만으로 "확인" 판정하면 안 됨

### OP-3: vendorName = 기관명은 정상
- vendorName이 "대구경북첨단의료산업진흥재단"인 경우가 많음
- 이는 **자체 집행**(기관이 직접 지출)을 의미하며, "불일치"가 아님
- 출장비의 경우 purpose에 출장자 이름이 있고, vendorName이 기관명이면 정상

### OP-4: 자문료 vendorName 판별
- 자문료의 vendorName이 기관명: 기관이 대신 지급 → 정상
- 자문료의 vendorName이 개인명: 직접 지급 → 정상
- purpose에 자문 대상자 이름이 있으면 증빙과 교차 확인

### OP-5: 위촉직 급여 충분 증빙
- 지출결의서 + 급여내역조회 + 위촉직급여정보 = **충분** → 적정
- 급여이체확인증, 개인별 급여명세서 = **추가 증빙** (선택)
- 이체확인증 미첨부만으로 "확인" 판정하면 안 됨

### OP-6: 인쇄비/수수료 증빙 판별
- evidenceType="기타"이고 견적서(2개) + 구매신청 + 구매검사 + 거래명세서가 있으면 → **적정**
- 거래명세서는 세금계산서가 없을 때 대체 증빙으로 인정
- 견적서가 2개 있으면 수의계약 절차 준수로 판단

### OP-7: 회의비 인원 판별
- 회의록/방명록이 첨부되어 있으면 **인원 확인 가능**으로 간주
- 파일 내용을 직접 볼 수 없으므로, 방명록 존재 = 인원 증빙으로 인정
- 방명록이 있는 상태에서 "참석인원 불명"으로 확인 판정하면 안 됨

### OP-8: 중복 데이터 일관성
- 동일한 파일 구성의 항목(예: R1=R21, R2=R22)은 **반드시 같은 판정**
- 데이터가 같으면 판정도 같아야 함

## 여비 (출장비)

- **필수**: 출장신청서(근무지외/내출장) + 지출결의서 + 교통비 영수증
- **관내출장** (통상 20,000원 수준): 출장신청서 + 지출결의서면 충분
- 출장자 이름과 신청서 이름 불일치 → **확인**
- 사업 목적과 무관한 출장 → **확인**
- 교통비 영수증: KTX 운임표, 택시 영수증, 하이패스 영수증, Receipt.pdf, 기차표 영수증 등
- IMG 파일(JPG/JPEG): 교통비 영수증 사진으로 인정
- vendorName이 기관명이고 purpose에 출장자 이름이 있으면 → 정상 (OP-3)

## 인건비

- **위촉직/기간제 급여**: 지출결의서 + 급여내역조회/위촉직급여정보 → **적정** (OP-5)
- **내부인건비 흡수**: 참여율 확인 필요 → **확인**
- 퇴직연금 관련 지출: 지출결의서 + 퇴직연금 납부서/증빙 → **적정**

## 자문료/원천세 (운영비/일반수용비)

- **필수**: 지출결의서 + 자문확인서 (또는 자문의견서)
- 자문자료/선정평가표 있으면 추가 증빙으로 적정 판정 강화
- 원천세는 자문료와 쌍으로 평가 (같은 증빙 공유)
- files=0 → **확인**
- vendorName이 기관명이어도 정상 (OP-4)

## 회의비 (운영비/일반수용비)

- **필수**: 지출결의서(또는 지출발의) + 회의록/방명록
- 회의록 미첨부 → **확인**
- 방명록 첨부 시 → 인원 확인 가능으로 간주 (OP-7)
- 보조금전용카드 결제 → 영수증 대체 가능

## 사무용품비 (운영비/일반수용비)

- **필수**: 구매신청 + 지출결의서 + 세금계산서(또는 보조금카드)
- 보조금전용카드 → 카드전표로 세금계산서 대체 (공통규칙3)
- 범용성 물품(PC, 노트북, 소프트웨어) → **확인**

## 임차료 (운영비)

- **필수**: 견적서 + 거래명세서/영수증 + 지출결의서
- 지출결의서 미첨부 → **확인**

## 용역비 (운영비/일반용역비)

- **필수**: 계약서 + 견적서(2개+) + 세금계산서 + 결과보고서
- 2천만원 초과 → 경쟁입찰 필요

## 수수료/인쇄비 (운영비/일반수용비)

- 세금계산서 또는 거래명세서 + 견적서 (OP-6)
- 보조금전용카드 결제 시 카드전표로 대체 가능
- 견적서2개 + 구매절차서류 + 거래명세서 → **적정**

## 공공요금 (운영비/공공요금및제세)

- 명세서/청구서 + 지출결의서 → **적정**
- 보험료: 보험증권 + 지출결의서 → **적정**

## 재료비 (운영비)

- 세금계산서/거래명세서 + 지출결의서
- 사업목적 부합 확인

## 대상자사례비

- 소득 지급명세서 (evidenceSub) → 시스템 내장 증빙 → **적정**
- files=0 + evidenceSub="" → **확인**
