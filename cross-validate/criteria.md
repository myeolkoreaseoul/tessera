# 비목별 판정 기준 (교차검증용)

> 대구경북첨단의료산업진흥재단 - 디지털헬스케어 의료기기 실증지원 사업
> 이 기관은 사업의 전문기관(수행기관)입니다.

## 공통 규칙

1. **0원 항목** → SKIP (취소전표)
2. **evidenceSub**에 "소득 지급명세서" 포함 → 시스템 내장 증빙 존재 (파일 없어도 증빙 인정)
3. **evidenceType**이 "보조금전용카드" → 카드전표로 세금계산서 대체 가능
4. **evidenceType**이 "전자세금계산서" 또는 "전자계산서" → 시스템 내장 결제증빙
5. **files=0개 + evidenceSub=""** → 증빙 없음 → **확인**
6. **보수적 원칙**: 확실한 경우만 "적정", 불확실하면 "확인"
7. "기타(첨부파일필수확인)" → 별도 첨부파일이 필요하다는 의미 → files 확인 필수

## 여비 (출장비)

- **필수**: 출장신청서(근무지외/내출장) + 지출결의서 + 교통비 영수증
- **관내출장** (통상 20,000원): 출장신청서 + 지출결의서면 충분
- 출장자 이름과 신청서 이름 불일치 → **확인**
- 출장복명서 미첨부 → **확인**
- 사업 목적과 무관한 출장 → **확인**
- 교통비 영수증(KTX, 택시 등) 있으면 추가 증빙으로 인정

## 인건비

- **위촉직/기간제 급여**: 지출결의서 + 급여내역/명세서 → **적정**
- **내부인건비 흡수**: 지출결의서 + 흡수 관련 서류 → 참여율 상세 확인 필요 → **확인**
- 근로계약서 필요 (미첨부 시 확인)
- 개인별 계좌 이체 확인

## 자문료/원천세 (운영비/일반수용비)

- **필수**: 지출결의서 + 자문확인서 (또는 자문의견서)
- 자문자료/선정평가표 있으면 추가 증빙으로 적정 판정 강화
- 원천세는 자문료와 쌍으로 평가 (같은 증빙 공유)
- files=0 → **확인**
- 자문수당 한도: 2시간 이하 20만원, 3시간 이하 30만원, 3시간 초과 40만원, 1일 한도 60만원

## 회의비 (운영비/일반수용비)

- **필수**: 지출결의서(또는 지출발의) + 회의록/방명록
- 회의록 미첨부 → **확인**
- 1인당 5만원 한도 초과 → **확인** (방명록 참석자 수로 확인)
- 보조금전용카드 결제 → 영수증 대체 가능
- 단일 기관 참여인력만으로 회의비 집행 → **확인**

## 사무용품비 (운영비/일반수용비)

- **필수**: 구매신청 + 지출결의서 + 세금계산서(또는 보조금카드)
- 고액(100만원+) → 견적서/구매검사 확인 필요 → 없으면 **확인**
- 범용성 물품(PC, 노트북, 소프트웨어) → **불인정** → **확인**

## 임차료 (운영비)

- **필수**: 견적서 + 거래명세서/영수증 + 지출결의서
- 견적서 2개 있으면 수의계약 절차 준수
- 지출결의서 미첨부 → **확인**
- 범용성 장비(노트북) 임차 → 사업목적 부합 확인 필요

## 용역비 (운영비/일반용역비)

- **필수**: 계약서 + 견적서(2개+) + 세금계산서 + 결과보고서
- 2천만원 초과 → 경쟁입찰 필요 (수의계약 시 사유서 필요)
- 계약기간이 사업기간 초과 → **확인**

## 수수료/인쇄비 (운영비/일반수용비)

- 세금계산서 또는 거래명세서 + 견적서
- 보조금전용카드로 결제 시 카드전표로 대체 가능

## 공공요금 (운영비/공공요금및제세)

- KT 명세서/청구서 + 지출결의서 → **적정**
- 보험료: 보험증권 + 지출결의서 → **적정**

## 재료비 (운영비)

- 세금계산서/거래명세서 + 지출결의서
- 사업목적 부합 확인
- 소모성 재료(시약, 시료 등) → 적정

## 대상자사례비

- 소득 지급명세서 (evidenceSub) → 시스템 내장 증빙 → **적정**
- files=0 + evidenceSub="" → **확인**
