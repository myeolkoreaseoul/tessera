# 보조금 공통 법규 정리 (상위법)

> 이 문서는 **보조금 관리에 관한 법률** 체계 하의 모든 사업에 공통 적용되는 규정을 정리한 것이다.
> 적용 시스템: e나라도움, 보탬e, 페이퍼정산 등
> 개별 사업의 운영세칙/지침이 이보다 구체적인 경우, 해당 지침이 우선한다.
> 실질적 심층 검토 기준은 `../common.md` (최상위) 참조.

> 4. 국가를 당사자로 하는 계약에 관한 법률 시행령 (대통령령 제35453호, 2025.4.22)

## 1. 보조금 사용 원칙 (보조금법 + 통합관리지침)

### 1.1 용도 외 사용 금지 (보조금법 제22조)
- 보조금을 **다른 용도**에 사용 불가
- 위반 시: 교부결정 취소 + 반환 + 이자 + 제재부가금(총액의 5배 이내)

### 1.2 보조사업비 카드 사용 원칙 (통합관리지침 제18조, 제19조)
- **보조사업비카드** 사용이 원칙 (전용 체크카드 또는 신용카드)
- 카드 사용이 불가한 경우: **전자세금계산서** 또는 증빙 첨부 후 계좌이체
- **별표1에 지정된 업종**(유흥업소 등)에서 카드 사용 불가
- 보조사업비카드 외 카드(개인카드, 법인카드 등) 사용 → **불인정 대상**
  - 단, 카드 발급 전 법인카드/개인카드 일시 사용은 예외적 허용

### 1.3 별도 계정 관리 (통합관리지침 제20조)
- 보조사업비는 **별도 계정(통장)**에서 관리
- 수익금 발생 시 별도 관리, 정산 시 반영

### 1.4 자료 보관 (통합관리지침 제31조)
- 관련 자료 **5년간 보관**

## 2. 계약 관련 규정 (통합관리지침 제21조, 21조의2 + 국가계약법 시행령)

### 2.1 일반경쟁 원칙
- **2천만원 초과** 물품/용역 구매 → 조달청/나라장터 통한 계약 원칙
- 물품/용역 가액이 2천만원 이하인 경우 → 수의계약 가능

### 2.2 수의계약 요건 (국가계약법 시행령 제26조, 통합관리지침 제21조의2)
수의계약 가능 금액 기준 (국가계약법 시행령 제26조 제1항 제5호):
- **2천만원 이하**: 물품의 제조·구매·임차계약 또는 용역계약
- **5천만원 이하**: 임대차 계약 (연액 또는 총액 기준)

수의계약 시 필수 요건:
1. **2개 이상** 업체로부터 견적서 제출받아야 함
2. 견적서를 **전자조달시스템** 이용하여 제출하도록 해야 함
   - 추정가격 2천만원 이하 또는 1인 견적 예외 사유에 해당하면 1개 가능
3. 집행금액의 **적정성** 확인 후 계약 체결
4. 수의계약 사유를 명시한 **사유서** 제출

### 2.3 견적서 관련 (국가계약법 시행령 제30조)
- 수의계약 시 **2인 이상**으로부터 견적서 수령 원칙
- 다음은 **1인 견적** 가능:
  1. 제26조 제1항 제1호 가목~나목, 같은 항 제2호, 제5호 가목7), 같은 호 마목~아목, 제27조, 제28조에 해당
  2. 추정가격 **2천만원 이하**
  3. 전자조달시스템 견적 제출 결과 제출자 1인뿐인 경우

### 2.4 계약서 작성 생략 (국가계약법 시행령 제49조)
- 계약금액 **3천만원 이하** 계약 → 계약서 작성 생략 가능

### 2.5 검사 (국가계약법 시행령 제55조)
- 계약 이행 완료 통지 후 **14일 이내** 검사 완료
- **검사조서** 작성 필요
- **3천만원 이하** 또는 매각계약 등 → 검사조서 작성 생략 가능 (제56조)

## 3. 정산보고서 불인정 기준 (정산보고서 작성지침 제5조)

### 3.1 공통 불인정 사유 (11가지)
1. **보조사업비카드 외 카드** 사용 (일부 예외 인정)
2. **사업기한 종료 후** 집행
3. **증빙 미비** 또는 증빙서류 위조·변조
4. **지출원인행위** 미완료 상태에서 지출
5. 전담기관/보조사업자의 **승인사항 미준수**
6. 보조금 **교부조건** 미준수
7. 보조사업과 **직접 관련 없는** 경비
8. **환급받을 수 있는 세금**(부가세 등)을 집행액에 포함
9. 보조금으로 **취득한 재산**의 목적 외 사용
10. **감사/조사**에서 지적된 사항의 미시정
11. 기타 관련 법령·지침에 위반되는 집행

### 3.2 주요 불인정 패턴 (실무에서 자주 발생)
- 사업비 통장에서 직접 집행하지 않은 건
- 개인카드/법인카드로 결제한 건 (보조사업비카드 아닌 경우)
- 회의록 미첨부
- 회의비 1인당 한도 초과
- 출장복명서 미첨부
- 범용성 장비(PC 등) 구입
- 참여인력에 대한 자문비 중복 지급

## 5. 정산 및 반환 (보조금법 제27조, 제31조 / 통합관리지침 제26조, 제27조)

### 5.1 실적보고 및 정산
- 보조사업 완료 또는 회계연도 종료 시 **실적보고서** 제출
- 집행잔액 + 이자 → **반납**
- 부가세 환급금 → 세외수입 처리

### 5.2 정산보고서 지연 시 삭감
| 지연 기간 | 삭감율 |
|----------|--------|
| 3개월 이내 | **10%** |
| 6개월 이내 | **20%** |
| 12개월 이내 | **50%** |

### 5.3 제재 (보조금법 제30조, 제33조, 제33조의2, 제40조, 제41조)
- **교부결정 취소**: 다른 용도 사용, 법령 위반, 거짓 신청
- **보조금 반환**: 취소 시 반환 + 이자
- **제재부가금**: 총액의 **5배 이내**
- **형사처벌**:
  - 거짓/부정 수단 → 10년 이하 징역 또는 1억 이하 벌금
  - 용도 외 사용 → 5년 이하 징역 또는 5천만원 이하 벌금

## 6. 중요재산 관리 (보조금법 제35조)

- 보조금으로 취득한 **중요재산**: 관리 의무
- 전담기관 승인 없이 양도, 교환, 대여, 담보 제공 금지

---

## 7. 대가 지급 (국가계약법 시행령 제58조)

- 검사 완료 후 계약상대자 청구일로부터 **5일 이내** 지급
- 지급 지연 시 → 지연이자 발생 (대출평균금리 기준)

## 8. 정산 검증 시 핵심 체크리스트

### 공통 사항
- [ ] 보조사업비 카드로 결제했는가?
- [ ] 사업기간 내 집행인가?
- [ ] 비목에 맞게 집행했는가?
- [ ] 증빙서류(영수증, 세금계산서, 계약서 등)가 구비되었는가?
- [ ] 환급 가능 세금(부가세)이 포함되지 않았는가?

### 계약 관련
- [ ] 2천만원 초과 → 경쟁입찰을 진행했는가?
- [ ] 수의계약 시 → 2개 이상 견적서가 있는가?
- [ ] 수의계약 사유서가 있는가?
- [ ] 계약서가 있는가? (3천만원 초과 시 필수)
- [ ] 검수조서가 있는가? (3천만원 초과 시 필수)

### 인건비
- [ ] 근로계약서 존재하는가?
- [ ] 참여율·참여기간 범위 내인가?
- [ ] 개인별 계좌로 이체했는가?

### 여비
- [ ] 출장복명서가 있는가?
- [ ] 여비지급 규정 범위 내인가?
- [ ] 식대 중복(회의비+출장비) 없는가?

### 회의비
- [ ] 회의록이 있는가?
- [ ] 1인당 한도를 초과하지 않았는가?
- [ ] 주류 등 유흥성 경비가 포함되지 않았는가?
