# 디지털헬스케어 의료기기 실증지원 사업 - 보조금 집행 기준 정리

> 출처: 디지털헬스케어 의료기기 실증지원 사업 운영세칙(전담기관-참여기관)_최종.pdf
> 전담기관: 대구경북첨단의료산업진흥재단 (K-MEDIhub)

## 1. 사업비 관리 원칙 (제9조, 제15조)

- 사업비는 **별도 계정**으로 관리, 보조금 계정과 연결된 **체크카드 또는 법인신용카드**로 집행
- 사업비 목적 외 사용 **엄격히 금지**
- **비목별** 출납상황 기록·관리
- 증빙서류: 결의서, 영수증, 견적서, 청구서, 계약서, 검수조서 등
- 사업비는 **사업 기간 이내**에 집행(지급결의, 계좌이체 또는 카드결제)
- 예치운증서(통장), 장부, 증빙서류 → 사업 종료 후 **5년간 보존**
- 자부담금: 총사업비(국비+지방비) 대비 **30% 이상** (현금 15% 이상 필수)

## 2. 비목별 예산 편성 기준 (별표1)

### 2.1 인건비 (110)

| 세목 | 내역 |
|------|------|
| 상용임금(03) | 무기계약직 |
| 일용임금(04) | 수개월/수일 임시직, 일용직보수, 기간제 근로자보수 |

**계상범위:**
- 근로계약서에 따라 지급하는 급여 및 상여금 총액
- 퇴직금 지급을 위한 퇴직충당금 적립액

**산정기준:**
- 인건비 = 실제월지급인건비 × 참여개월 × 참여율(참여율 = 연봉총액 중 사업 참여 비율)
- 협약 당해연도 기획재정부 계약예규 학술연구용역인건비 기준 단가 참조 가능
- **총 사업비(국비+지방비)의 30% 이하**로 계상
- 30% 초과 시 → 전담기관 승인 필요 (사유서 및 공문 제출)
- 국고보조금 참여 인력 인건비에는 **기관부담 4대보험 및 퇴직급여충당금 포함**

### 2.2 운영비 - 일반수용비 (210-01)

| 항목 | 내역 |
|------|------|
| 1. 사무용품 구입비 | 필기용구, 각종용지 등 사무용 제 잡품 |
| 2. 인쇄비 및 유인비 | 자료, 보고서, 책자, 각종 양식, 전단 등 |
| 3. 안내·홍보물 등 제작비 | 현수막, 간판 등 행사 안내 및 홍보용 물품 |
| 4. 소모성 물품 구입비 | - |
| 5. 간행물 등 구입비 | 신문·잡지·관보·도서·팸플릿 등 정기/비정기 |
| 6. 비품 수선비 | 책상, 의자, 캐비닛, 전산기기 등 사무용 비품 수선 |
| 7. 각종 수수료 및 사용료 | 물품관리위탁수수료, 업무대행수수료, 외국환대체송금, 전송금, 등기·소송료, 검정료, 감정료, 시험료, 회계검사수수료, 물품 보관·운송료, 고속도로통행료, 주차·차고료 등 |
| 8. 업무위탁대가 및 사례금 | 변호료·수임료, 속기·원고측량 용역대가, 전문가 자문료, 현상 모집 상금, 조직업무 조력자 사례금, **회의참석사례비 및 안건검토비** |
| 9. 공고료 및 광고료 | TV·신문·잡지 기타 간행물 광고 |
| 10. 각종 회의비, 전문가 활용비 | 아래 별도 정리 |
| 11. 행사지원에 따른 경비 | - |
| 12. 기타 소규모 물품 구입 및 용역 | - |

#### 회의비 집행 시 증빙서류
- **내부결재문서** (회의목적, 일시, 장소, 대상, 산출내역 포함)
- **회의록** (또는 회의결과보고서) 및 **방명록(서명포함)**
- 주말/공휴일 집행 시 → 근무 또는 사업수행 확인 서류 및 증빙자료
- 쇼핑몰(독립적 소매집포가 모인 상업장소) 사용 시 → 거래 내역 확인 가능한 **증빙(영수증)** 첨부

#### 회의비 주의사항
- 동일 기관 사업추진비 집행 시 **과제 미참여인력 참석 필수**
- 참석자 서명 있는 회의록에 소속 성명 기재, 과제 참여/미참여 인력 구분 가능하도록
- 동일 컨소시엄 내 주관-참여기관 업무추진/협의 회의 → 참여기관이 아닌 **주관기관**에서 예산 편성·집행
- **회의비 1인당 50,000원 초과 불가**

#### 전문가 활용비 지급기준
- 사업 참여인력에 지급하는 자문료 및 수당 **불인정**
- 전문가 활용비는 지급기준을 준용하여 집행

| 구분 | 단가 |
|------|------|
| 자문수당 (2시간 이하) | 200,000원/명 |
| 자문수당 (3시간 이하) | 300,000원/명 |
| 자문수당 (3시간 초과) | 400,000원/명 |
| 초과 시 시간당 | 50% 가산 (1일 1회 한함, **1일 한도 60만원**) |

> ※ 공직자(교직원, 언론인 제외)일 경우 부정청탁 및 금품 등 수수의 금지에 관한 법률 참조

### 2.3 운영비 - 공공요금 및 제세 (210-02)

- 우편요금, 전신(전보)·전화요금, 모사전송기 등 회선 사용료
- 철도화물 운송요금
- 전기·가스료, 상·하수도료
- 법령에 의한 제세(자동차세 포함) 및 국내부담금, 협회비, 기타 계약에 의한 각종 부담금
- 소송 공탁금과 국고제당금
- 임대차 계약에 의한 보증금 및 전세금
- 보험계약에 의한 각종 보험료
- 에너지 절약 성과배분계약에 따른 설비투자 상환금

### 2.4 운영비 - 임차료 (210-07)

- 임대차계약에 의한 토지, 건물, 시설, 장비, 물품 등의 임차료
- 장소, 건물 등의 일시 임차료
- 각종 시설 및 장비의 리스료
- 물건 보관용 간단한 창고 이용료
- 버스·승용차 등의 차량 임차료
- ASP 서비스 이용에 따른 임차료

> ※ **자산성 장비 및 물품은 임차료를 사용하여 렌탈로 진행할 것** (210-07목에서 집행)

### 2.5 운영비 - 시설장비유지비 (210-09)

- 건물 및 건축설비 유지 관리비
- 통신시설 및 기상관측장비 유지비 (대체비는 노임, 제비용 포함)
- 원동기 등 동력장치 중장비 등에 소요되는 유류대, 기타 육상 운반구(차량제외) 유지비
- 시설장비유지관리의 용역비(노무비와 제비용 포함)
- 내용연수를 현저히 증가시키는 대규모 수리비는 → **시설비 목으로 계상**

### 2.6 운영비 - 재료비 (210-11)

- 사업용 및 시험연구, 실험·실습 등에 소요되는 **소모성 재료비**
  - 실험·실습기자재, 시약, 시료 구입비
  - 직접제작 또는 시공하는 기계·기구, 선박, 기타 공작물 및 건물에 소요되는 재료비
- 제품생산에 소비되는 각종 재료비용 (주요 재료비, 보조 재료비, 매입부품비, 소모공기구비품비로 구분)
- 광물 및 기타 특수한 물건의 구입비
- 동물, 식물 및 식물종자 구입비
- 사료구입비

### 2.7 운영비 - 일반용역비 (210-14)

- 기관의 업무추진 과정에서 전문성이 필요한 행사운영, 채용, 영상자료 제작 등의 일반 업무를 용역계약을 통해 대행시키는 비용
- 협약 시 사업계획서에 명시한 용역 외 **추가 용역 계약 시 전담기관 승인(공문) 필요**

**계약 기준 (국고보조금 통합관리지침 제21조, 21조의2):**
- **2천만원(공급가액 기준) 초과** 물품 및 용역구매 → **일반경쟁** 원칙
- 수의계약 시:
  - (1) 사유를 명시한 사유서를 공문과 함께 제출
  - (2) 사전에 **최소 2개 이상** 업체로부터 견적을 받고
  - (3) 집행금액의 적정성 등을 확인한 후 계약 체결
- 수의계약 시 해당 업체를 최종 선정한 **내부평가 문서** 존재 필요
- 경쟁입찰 대상이지만 수의계약 진행 시 → **최소 2개 이상** 다른 업체 추가 견적 필요

**불인정 사례:**
- 외주용역 계약기간이 **사업기간을 초과**하는 경우
- 용역수행기관이 참여기관(컨소시엄)의 **관계회사 또는 경영자의 친인척**인 경우

### 2.8 여비 (220)

| 세목 | 내역 |
|------|------|
| 국내여비(01) | 각 기관이 정한 기준에 따른 실 소요 경비 |
| 국외여비(02) | 각 기관이 정한 기준에 따른 실 소요 경비, 외빈초청 여비(숙식비 및 항공료 등 교통비) |

> ※ 국외출장 및 외빈 초청 건은 **전담기관 승인** 필요
> ※ 출장 여비는 참여기관의 여비지급 기준에 따르며, 기준이 없으면 **공무원 여비규정** 준용

## 3. 불인정 기준 (별표2)

### 3.1 공통사항

| 불인정 사유 | 비고 |
|-----------|------|
| 사업비 통장/계정에서 직접 집행하지 않은 경우 | 인건비 지급, 출장비 지급, 수입구매(외화송금), 외국인 전문가 활용비 등 부득이한 경우는 인정 |
| 사업비 이외의 카드를 사용한 경우 | 사업비카드 발급 전 법인카드 및 개인카드 사용은 일시적으로 허용 |
| 사업기간 종료 후 집행, 증빙 미비 | - |
| 컨소시엄 주관-참여기관 간의 집행 | 재료, 임차, 해당 참여기관 소속 인건비, 전문가수당 등 지급은 예외 |
| 환급받을 수 있는 세금(부가세 등)을 집행액에 포함 | - |
| 사업비 변경 등 전담기관 승인사항 미준수 | - |
| 전담기관/위탁정산기관의 증빙서류 제출요청에 불응 | - |
| 기타 전담기관이 정상적 사업비 집행으로 간주하지 않는 경우 | - |

### 3.2 인건비 불인정

| 불인정 사유 |
|-----------|
| 참여연구원 변경을 전담기관에 승인/통보하지 않고 인건비 지급 |
| 참여연구원 이외 지원인력의 급여를 인건비로 집행 |
| 개인 참여율 및 참여기간을 초과하여 집행 |
| 개인별 계좌로 이체하지 않은 금액 |
| 인건비를 다른 용도로 집행 |

### 3.3 여비 불인정

| 불인정 사유 |
|-----------|
| 과제와 관련 없는 개인 목적 국내외 여비 |
| 참여연구원이 아닌 인원에게 지출된 여비 (사업 관련 외부전문가 여비 인정 가능) |
| 출장비에 식대 포함된 경우 회의비(식대) 중복 사용 시 해당 식대 |
| 국외여비 지급일 또는 결제일의 매매기준율 미적용 시 그 차액 |
| 국내외여비에 포함되는 숙박비/식비 등을 출장지 관계기관에서 지급한 경우 |
| 출장복명서가 없거나 출장목적이 명확하지 않은 여비 |
| 출장기간 초과, 체재지역 외 사용액 |
| 여비지급규정 초과액 |

### 3.4 일반수용비 불인정

| 불인정 사유 |
|-----------|
| 사업과 무관한 지출 (참여기관 자체 홍보를 위한 광고료) |
| 참여기관 내부보유 장비, 시설, 공간의 임차료를 지불한 경우 |
| **범용성 장비(PC, 주변기기, 복사기, 가구 등 사무용기기, 집기) 구입 또는 대여, 유지관리비** |
| **범용성 소프트웨어(사무처리 프로그램, 바이러스 백신 등) 구매** |
| 참여기관 소속인원 또는 인건비가 계상된 인원에 대한 **자문비 및 수당** |
| 단일 참여기관 참여인력만으로 회의비 집행 |
| **주류 등 유흥성 경비** 포함 |
| 경조사비 및 선물구입비 |
| **회의록 없이 집행** |
| **회의비 1인당 5만원 초과** |
| 사업수행에 직접적으로 관련 없는 경비 (사무기기/연구장비/시설 유지보수, 기념품, 선물 구입비, 기관 공통 간행물 구독, 종신 학회비 등 개인성 경비) |
| 전문가 활용비 초과 지급에 대해 집행근거(경력, 활동 등 사유서)를 미제출한 경우 |

## 4. 주요 금액 한도 정리

| 항목 | 한도 |
|------|------|
| 인건비 | 총 사업비(국비+지방비)의 **30% 이하** |
| 회의비 | **1인당 5만원** 이하 |
| 전문가 자문수당 (2h 이하) | **20만원/명** |
| 전문가 자문수당 (3h 이하) | **30만원/명** |
| 전문가 자문수당 (3h 초과) | **40만원/명** |

## 5. 비목 간 변경 (제11조 ⑥)

- 최초 사업계획 산출내역서 상 비목 간 비용의 변동 및 비목과 세목의 신설 → 전담기관 사전승인
- 비목의 **30% 이상** 해당하는 금액의 세목 간 변동 → 전담기관 사전승인(공문) 필요
  - 비목의 30% 미만에 해당하는 세목 간 변경 → 전담기관 사전승인 없이 내부 결재 후 사용 가능 (단, 사전 협의 후 e나라도움에서 승인 요청)

## 6. 심층 검토 보완사항 (대구경북재단 180건 검토 경험, 2026.02)

> 공통 심층 검토 기준은 `guidelines/common.md` §9 참조. 아래는 이 사업에만 해당하는 특수사항.

### 6.1 전문가활용비 교통비 합산 — 비목분류 확인

- 세칙 별표2 여비: "외부전문가 교통비 지급 인정" (여비 비목에서)
- 세칙 별표1 일반수용비 항목10: "전문가 활용비는 전담기관 내부 기준 준용"
- 실무: 자문수당 + 교통비를 **전문가활용비(일반수용비)**에 합산 집행하는 사례 존재
- 여비 비목에서 별도 집행하는 것이 세칙 원칙이므로, 합산 집행 시 **비목분류 확인 보완요청** 가능

**보완요청 문구:**
> "교통비영수증(KTX) 및 지출발의서상 자문수당 ○○원과 교통비가 합산 집행된 것을 확인하였습니다. 다만, 외부전문가 교통비는 여비 비목에서 별도 집행하는 것이 원칙이므로(세칙 별표2 여비 참조), 전문가활용비에 교통비를 합산 집행한 사유를 확인하여 주시기 바랍니다."

### 6.2 전문가활용비 초과 — 불인정 기준 주의

- 별표2: "전문가 활용비 초과 지급에 대해 집행근거(경력, 활동 등 **사유서**)를 **미제출**할 경우" 불인정
- 즉, **초과 자체가 불인정이 아님**. 사유서 미제출이 불인정.
- 60만원 초과 시: 사유서 첨부 여부 확인 → 있으면 적정, 없으면 확인

### 6.3 전담기관(대구경북재단) 자체 정산 특수성

- 구조: 보건복지부(주관) → KHIDI(총괄) → **대구경북재단(전담)** → 참여기관
- 세칙 제2조: "사업운영기관과 **모든 참여기관**에 적용" → 대구경북재단 자체에도 적용
- 전담기관 자체 정산은 "검찰이 경찰청을 감사하는 것"과 유사 — 근거를 명확히 해야 함
- 지침 제10조⑤-2: "전담기관은 인프라운영 및 지원을 위한 예산 중 **30% 이하로 인건비** 계상 가능"
- 내부인건비 흡수, 원천세 등 전담기관 특유의 집행 패턴에 주의

### 6.4 인건비 상한

- 별표1: 자기부담금을 제외한 총 사업비(국비+지방비)의 **30% 이하**
- 30% 초과 시: 사유서 및 공문을 제출하여 전담기관 승인 필요
- 전담기관 자체 정산 시: 지침 제10조⑤-2에 의해 인프라운영 예산 중 30% 이하
