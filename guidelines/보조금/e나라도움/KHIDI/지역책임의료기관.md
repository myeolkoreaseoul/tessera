# 지역책임의료기관 사업 - 보조금 집행 기준 정리

> 출처: 「2025년 책임의료기관 통합 사업 안내」 5.6절(예산 집행 기준)

---

## 비목별 상시점검 검증 프레임워크

### 공통 검증 (모든 비목)

| 검증 항목 | 기준 | 판단 |
|-----------|------|------|
| 증빙서류 존재 | 1개 이상 첨부 필수 | 미첨부 → 부적정 |
| 증빙서류 유형 | 세금계산서, 신용카드매출전표, 금전등록기영수증, 온라인지급증빙서 중 1개 이상 | 유효 증빙 없으면 확인필요 |
| 금액 일치 | 증빙상 금액이 집행금액 또는 공급가액과 일치(±5%) | 불일치 → 확인필요 |
| 거래처 일치 | 증빙상 사업자/거래처명이 엑셀 거래처와 일치 | 불일치 → 확인필요 |
| 사업 연관성 | 집행용도가 책임의료기관 사업(공공보건의료 협력체계)과 관련 있는지 | 무관해 보이면 확인필요 |
| 상품권/유가증권 불가 | 상품권, 쿠폰, 현금성 유가증권 구매 불가 | 해당시 → 부적정 |

### 1. 인건비 (110)

- **대상 키워드**: 급여, 보수, 인건비, 퇴직금, 상여금, 초과근무수당
- **필수 증빙**: 급여명세서 + 통장사본(이체내역)
- **전담인력 한정**: 전담인력(또는 사업계획서상 겸임의사)만 가능
- **타 인건비 중복 불가**: 타 사업 인건비와 중복 지급 불가
- **월별 지급 원칙**: 분기 일괄이체 부적절
- **퇴직금**: 사업수행기간분만 가능
- **겸임 불가 원칙**: 전담인력 겸임 업무 원칙 불가 (불가피시 복지부 승인)
- **주의 키워드**: 당직수당, 타부서 발령, 겸임 → 확인필요

### 2. 운영비 - 일반수용비 (210-01)

#### ① 사무용품/소모성 물품
- **필수 증빙**: 세금계산서 또는 카드영수증
- **취득단가 한도**: 50만원 미만 (50만원 이상 → 부적정, 자산취득비로 계상 필요)
- **상품권 불가**: 상품권, 쿠폰 등 현금성 유가증권 → 부적정
- **환자 지급 물품 불가**: 환자에게 직접 지급 물품 → 부적정
- **사업 관련성**: 사무용품, 전산소모품, 회의용품 등 사업수행 관련만

#### ③ 안내·홍보물 제작비
- **총액 한도**: 총 5,000만원 이하 (누적 확인)
- **단가 한도**: 홍보물품 단가 5만원 이하 (청탁금지법)
- 5만원 초과 → 부적정

#### ④ 자문수당
- **필수 증빙**: 자문수당지급조서
- **총액 한도**: 내부 인력 대상 총 2,000만원 이하 (누적 확인)
- **건당 단가**:
  - 각종 자문: 2시간까지 150,000원 / 초과 200,000원 / 서면 100,000원
  - 심포지엄 발표: 2시간까지 300,000원 / 초과 400,000원
- **겸임수당 의사 중복 불가**: 겸임수당 지급 의사에게 내부 자문수당 중복 → 부적정

#### ⑤ 협력 인센티브
- **총액 한도**: 내부 협력인센티브 총 5,000만원 이하 (누적 확인)
- **건당 단가**: 1건당 5만원 이하 / 초과시 → 부적정
- **네트워크 수당**: 외부인력, 1인당 월 50만원 이하
- **중복 지급 불가**: 자문수당, 교육수당과 동일건 중복 → 부적정

#### ⑥~⑨ 기타 일반수용비
- 간행물: 사업 필요 수량만, 대량 구입 불가
- 비품 수선비: 대규모 수선(공사) 불가
- 수수료/사용료: 사업수행 관련만
- 공고료/광고료: 사업 대상층과 매체 적합성 확인

### 3. 운영비 - 기타 세목

#### 피복비 (210-03)
- 전담인력 근무피복(가운 등)만 가능

#### 임차료 (210-07)
- 장소/물품/차량 임차
- **차량임차 제한**: 환자이송 목적 불가, 차량유지비(세차/보험) 불가
- 사업수행기간(해당연도) 임차료만

#### 유류비 (210-08)
- **총액 한도**: 총 300만원 이하 (누적 확인)
- **임차차량 한정**: 임차료(210-07)로 임차한 차량에만 집행 가능
- 비임차차량 → 부적정

#### 복리후생비 (210-12)
- 4대보험 기관부담금 등, 전담인력에 한함

#### 교육훈련비 (210-16)
- **총액 한도**: 총 300만원 이하 (누적 확인)
- **전담인력 한정**: 전담인력의 외부교육만 / 비전담인력 → 부적정
- **사업 관련성**: 사업과 직접 관련있는 교육만 인정

### 4. 여비 (220)

- **필수 증빙**: 출장명령서 + 출장보고서
- **숙박비 증빙**: 세금계산서/카드매출전표/현금영수증
- **운임 증빙**: 승차권 등
- **기준**: 공무원 여비업무 처리기준
- **대상**: 전담인력 + 사업참여 내·외부인력

### 5. 업무추진비 (240)

- **필수 증빙**: 카드영수증 (식음료)
- **인당 한도**: 최대 5만원 / 초과시 → 부적정
- **총액 한도**: 보조금 총액의 5% 이하 (누적 확인)
- **상품권 불가**: 상품권/쿠폰 등 현금성 유가증권 → 부적정
- **대면 원칙**: 대면 참석자 대상만
- **내부회의 제한**: 외부 인사 미참여 내부회의 원칙 불가 (원내 대표·실무협의체/간담회 예외)
- **사업수행인력만 회의**: 겸임·전담인력·관리직만 참여한 회의 경비 → 부적정

### 6. 연구용역비 (260)

- **필수 증빙**: 계약서 + 세금계산서
- **위탁용역비 한도**: 총 6,000만원 이하 (누적 확인)
- **공공의료연계망**: 국립중앙의료원 위탁, 기관당 500만원 이상
- **사업계획서 기재**: 필요성 및 상세내역 기재 필수
- **동일기관 위탁 불가**: 원내 위탁 불가 (내부 자문비로 활용) → 부적정

### 7. 유형자산 (430)

- **필수 증빙**: 세금계산서 + 자산취득보고서
- **한도**: 계속사업기관 500만원 (누적 확인)
- **구매가능 품목**: 지침 명시 품목만 (책상, 노트북 등)
- **사업계획서 명시**: 사전 보고되지 않은 품목 불인정

### 8. 회의참석사례금 (일반수용비 내)

- **대상**: 외부 위원 협의체/위원회 참석
- **건당 단가**: 2시간까지 150,000원 / 초과 200,000원
- **1일 1건 제한**: 동일인 1일 다건 사례금 → 부적정

### 검증 프로세스 (각 건당)

```
Step 1: 기본 정보 확인
  → 비목명/세목명으로 해당 비목 규칙 매핑
  → 집행용도, 품명으로 세부 카테고리 특정

Step 2: 증빙서류 검증
  → 파일 존재 여부 (0개 = 부적정)
  → 문서 유형 분류 (세금계산서/카드영수증/지출결의서/급여명세 등)
  → 비목별 필수 증빙 존재 여부

Step 3: 금액 검증
  → 증빙상 금액 추출
  → 집행금액/공급가액과 대조
  → 비목별 건당 한도 초과 여부 (자문수당 20만원, 업무추진비 5만원, 소모품 50만원 등)

Step 4: 내용 적정성 검증
  → 거래처명 일치
  → 사업 연관성 (집행용도 키워드 vs 증빙 내용)
  → 비목별 금지사항 해당 여부
  → 비목 설정 적절성 (이 지출이 이 비목에 맞는지)

Step 5: 누적 한도 검증 (별도 집계)
  → 자문수당 총 2,000만원
  → 안내홍보물 총 5,000만원
  → 협력인센티브 총 5,000만원
  → 유류비 총 300만원
  → 교육훈련비 총 300만원
  → 연구용역비 총 6,000만원
  → 유형자산 총 500만원 (계속사업기관)
  → 업무추진비 총 보조금의 5%

Step 6: 종합 판단
  → 적정: 필수증빙 존재 + 금액일치 + 한도이내 + 사업관련성 인정
  → 확인필요: 금액불일치, 증빙부족, 한도근접 등
  → 부적정: 증빙미첨부, 한도초과, 금지항목, 비목부적절
```
