# 보탬e 시스템 공통

> 보탬e(losims.go.kr) — 지방보조금 정산시스템
> 상위: `../../common.md` (정산 공통) → `../common.md` (보조금 공통)

---

## 1. 시스템 특성

- URL: `https://www.losims.go.kr/lss.do`
- UI 프레임워크: cl-* (커스텀, 모든 요소가 div 렌더링)
- 데이터 통신: WebSocket (XHR 인터셉트 불가)
- **5분 세션 타임아웃** — 반드시 주기적 새로고침

## 2. 네비게이션

- 정산관리 → 민간회계사정산검토 → 검토대상 목록조회(민간회계사)
- 회계연도 선택 → 보조사업자명 검색 → "작성중" 클릭 → 집행내역

## 3. 사업별 하위 디렉토리
