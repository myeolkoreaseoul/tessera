# R&D혁신법 공통

> 국가연구개발혁신법 체계 하의 모든 사업에 공통 적용되는 규정.
> 상위: `../common.md` (정산 공통)
> 적용 시스템: 이지바로, RCMS 등
> **법적 근거**: 국가연구개발혁신법, 국가연구개발사업 연구개발비 사용 기준(과기부고시 제2025-9호, 2025.3.6. 시행)

---

## 1. 비목 체계

혁신법 R&D 과제는 다음 비목 체계를 사용한다 (보조금법 체계와 다름):

| 대비목 | 세비목 |
|--------|--------|
| **인건비** | 내부인건비(비영리), 내부인건비(영리), 외부인건비, 연구근접지원인력, 학생인건비 |
| **연구장비재료비** | 연구시설장비비, 연구재료비 |
| **연구활동비** | 출장비(국내/국외), 회의비, 전문가활용비, 연구개발서비스활용비, 교육훈련비, 연구실운영비, 야근/특근식대, 소프트웨어활용비, 클라우드컴퓨팅서비스, 그밖의비용 |
| **연구수당** | (단일) |
| **보안수당** | (단일, 보안과제만 해당) |
| **간접비** | 연구지원인력비, 성과활용지원비, 연구활동지원금(비영리만) 등 |
| **위탁연구개발비** | (단일) |

## 2. 판정 원칙

1. **문서 내용 종합 판단** — 파일명이 아닌 OCR 내용으로 문서 종류를 판단
   - "품의서", "지출결의서", "구매요청서" → 내부결재문서로 인정
   - "원천징수영수증" → **연간 총급여 수준 확인용** (월별 지급 사실 확인 불가)
   - 월별 인건비 지급 확인: **급여명세서(월별) + 계좌이체증명**이 원칙
2. **보수적 판단** — 확실한 경우만 "점검완료", 나머지는 "보완요청"
3. **연구비카드 또는 계좌이체 원칙** (고시 제22조⑤) — 현금 사용 시 사유 소명 요구
4. **부가세 환급 가능분 불인정** (고시 제21조④1) — 모든 기관 적용
5. **영리/비영리 기관유형 구분 필수** — 요구 서류가 다름

---

## 3. 영리/비영리 주요 차이

| 항목 | 비영리 (대학, 정부출연연, 기타비영리) | 영리 (기업) |
|------|--------------------------------------|-------------|
| 인건비 원칙 | 현물 원칙 (교원 등) | 현물 원칙, 현금 시 지급요건 소명 |
| 필수 추가서류 | - | 건강보험자격득실, 별지3호(현금인건비), 별지6호(현물) |
| 연구실운영비 | 내부규정 필요 | 별지4호+사전승인 필수, 소모성비용 불가 |
| 연구활동지원금(간접비) | 허용 | **불허** (고시 제31조③) |
| 부가세 | 환급 가능분 환원 | 환급 가능분 환원 (더 빈번) |

---

## 4. 기간 관련 주의사항

- **연구시설장비비**: 과제 종료 **2개월 전**까지 구입 및 입고 완료 (고시 제23조⑤4)
- **연구재료비**: 종료 **1개월 전** 다량 구매 시 사유 소명 요구
- **소프트웨어**: 최소단위 계약 시 기간 초과분 인정 가능 (고시 제25조⑩, 소명 필수)
- **연구수당**: 단계 종료 후 **1개월 이내** 지급 필수 (고시 제26조③)
- **연구시설장비 3천만원 이상**: ZEUS 등록 + 사전승인 필수 (고시 제23조④, 제73조①5)

---

## 5. 불인정 vs 보완요청 구분

**명백한 불인정 (환원 대상):**
- 미참여 연구원 인건비
- 외부인원 없는 회의비 **식비 집행분**
- 영리기관 연구실운영비 소모성비용
- 연구수당 1인 70% 초과분
- 연구수당 직접비 대비 20%p 초과분
- 영리기관 연구활동지원금
- 비참여연구자·비연구근접지원인력 교육훈련비

**증빙 보완 가능:**
- 나머지 대부분 (증빙 미비, 서류 누락 등)

## 6. 공통 보완요청 사항 (전 비목)

| 상황 | 보완요청 문구 |
|------|-------------|
| 부가세 환원 | 집행금액에 환급 가능한 부가가치세가 포함되어 집행된 것으로 보입니다. 부가세를 환원 조치 바랍니다. (고시 제21조④1) |
| 연구비카드 미사용 | 연구개발비는 연구비카드 또는 거래처에 대한 계좌이체 형태로 사용함을 원칙으로 합니다. (고시 제22조⑤) 해당 규정 미준수에 대한 사유를 소명 바랍니다. |
| 과제 연관성 소명 | 해당 과제와의 직접적인 연관성을 소명 바랍니다. 또한 연구개발계획서 업로드 바랍니다. |
| 간접비성 경비 직접비 집행 | 전기료, 통신료, 관리비 등은 간접비로 사용하는 것이 타당합니다. 합리적 산출 자료가 있는 경우 업로드, 없는 경우 환원 처리 바랍니다. |
| 특허출원비 비목 오류 | 특허출원비 등 지식재산권 출원·등록 비용은 '간접비-성과활용지원비'로 집행해야 합니다. 비목 변경 바랍니다. (고시 제5조②) |
