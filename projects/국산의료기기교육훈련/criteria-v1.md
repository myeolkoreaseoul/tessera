## 1. 공통 규칙

1. **0원 항목** → SKIP (취소전표)
2. **evidenceSub**에 "소득 지급명세서" 포함 → 시스템 내장 증빙 (파일 없어도 인정)
3. **evidenceType = "보조금전용카드"** → 카드전표로 세금계산서 대체 가능
4. **evidenceType = "전자세금계산서"/"전자계산서"** → 시스템 내장 결제증빙
5. **files=0 + evidenceSub=""** → 증빙 없음 → **확인**
6. **보수적 원칙**: 확실한 경우만 "적정", 불확실하면 "확인"
7. **"기타(첨부파일필수확인)"** → 별도 첨부파일 필수 확인

## 3. 심층분석 규칙 (DR-1~9 — analysis 필드 기반)

> deep-analyze.js가 각 행에 `analysis: { flags: [...], fields: {...} }` 추가.
> 아래 규칙은 analysis.flags를 참조하여 판정에 반영.

### DR-1: 원천세 납부증빙

- **플래그**: `원천세_납부증빙_미첨부`
- **조건**: analysis.fields.type = "원천세" + analysis.fields.taxPaymentProof = false
- **판정**: → **확인**
- **보완요청**: "원천세 집행건에 대해 국세청 납부 여부를 확인할 수 있는 증빙(원천징수이행상황신고서 등)이 첨부되어 있지 않습니다. 해당 원천세가 실제 납부되었음을 확인할 수 있는 서류를 제출하여 주시기 바랍니다."
- **근거**: 원천세는 월별/반기별 일괄 신고·납부. 공제만 하고 미납 가능성 있음.

### DR-2: 내부인건비 흡수

- **플래그**: `내부인건비_산출근거_미비`
- **조건**: analysis.fields.type = "내부인건비흡수" + analysis.fields.missingDocs.length > 0
- **판정**: → **확인**
- **보완요청**: "내부인건비 흡수 집행건에 대하여 인건비 산정기준(실제월지급인건비 × 참여개월 × 참여율)의 적정성을 확인할 수 있도록 다음 서류를 제출하여 주시기 바랍니다. 1) 사업 참여인력 명단 및 개인별 참여율 2) 참여인력의 급여대장 3) 기관부담 4대보험료 산출내역"
- **근거**: 구조는 합법이나 산출근거 없으면 적정성 확인 불가. [근거: 별표1 인건비 산정기준]

### DR-3: 자문료 비목분류

- **플래그**: `자문료_교통비합산_영수증있음`
- **조건**: analysis.fields.estimatedTransport > 0 + 교통비영수증 있음
- **판정**: → **적정** (단, 비목분류 보완요청)
- **보완요청**: "교통비영수증(KTX) 및 지출발의서상 자문수당과 교통비가 합산 집행된 것을 확인하였습니다. 다만, 외부전문가 교통비는 여비 비목에서 별도 집행하는 것이 원칙이므로, 전문가활용비에 교통비를 합산 집행한 사유를 확인하여 주시기 바랍니다."

- **플래그**: `자문료_한도초과_교통비영수증_없음`
- **조건**: 세전 > 한도 + 교통비영수증 없음
- **판정**: → **확인**

### DR-4: 출장복명서

- **플래그**: `출장복명서_미첨부`
- **조건**: 여비 건 + 출장복명서(결과보고) 없음
- **판정**: → **확인** (이 사업은 출장복명서 필수)
- **근거**: 별표2 여비 불인정: "출장복명서가 없거나 출장목적이 명확하지 않은 여비" → 불인정 [근거: 별표2 3.3]
- **주의**: 출장신청서 ≠ 출장복명서. 통합 서식일 경우 OCR에서 "복명" 키워드 확인.

### DR-5: 재원변경 중복

- **플래그**: `재원변경_중복건`
- **조건**: analysis.fields.duplicateOf 존재
- **판정**: → **SKIP** (원본건만 검토)
- **근거**: 원래 집행 취소 → 재원 변경 → 재집행. 이중 계상 아님.

### DR-6: 공용첨부파일 미수집

- **플래그**: `공용첨부파일_미수집`
- **조건**: files=0 + 공용첨부 ID 존재
- **판정**: → **PENDING** (수동 확인 필요)

### DR-7: 부가세 포함 집행

- **플래그**: `부가세_포함_집행`
- **조건**: vat > 0 + totalAmount = supplyAmount + vat
- **판정**: → **확인** (영리법인 매입세액공제 가능분)
- **보완요청**: "부가가치세가 집행액에 포함되어 있습니다. 환급 가능한 매입세액이 포함된 경우 불인정 대상이므로 확인하여 주시기 바랍니다." [근거: 별표2 공통]

### DR-8: 2천만원 초과 경쟁입찰

- **플래그**: `2천만원초과_경쟁입찰_확인필요`
- **조건**: 물품/용역 공급가액 > 2천만원
- **판정**: → **확인** (경쟁입찰 절차 확인 필요)

### DR-9: 증빙 완전 부재

- **플래그**: `증빙_완전부재`
- **조건**: files=0 + 시스템 내장 증빙 없음 + evidenceSub 없음
- **판정**: → **확인**

## 4. 사업별 특화 규칙 (SR- prefix)

### SR-1: 회의비 인당 30,000원 한도 (이중기준 — 30,000원 적용)

- **적용 기준**: 별표2(불인정 기준)에서 **1인당 30,000원 초과 시 불인정** 명시
- **주의**: 별표1의 50,000원은 편성(계획) 한도이며, 실제 집행 불인정 기준은 별표2의 30,000원
- **판정**: 1인당 집행액 > 30,000원 → **확인**
- **추가 불인정**: 회의록 없음 / 원격지 회의비 / 정당한 사유 없이 휴일 집행 → **확인**
- **근거**: 별표1(편성기준 50,000원), 별표2 직접비 불인정 "회의비는 1인당 3만원을 초과할 수 없음" [근거: 운영지침 별표2 3.4]

### SR-2: 유형자산취득비 — 정부지원금 40% 한도

- **조건**: 유형자산취득비 누계 > 정부지원금 × 40%
- **판정**: → **확인** (한도 초과 여부 확인 필요)
- **확인 항목**: 관리기관 승인 여부, 누계 비율
- **근거**: 운영지침 "유형자산 취득비용으로 국고보조금의 40%를 초과할 수 없음" [근거: 별표1 2.7 유형자산취득비]

### SR-3: 교육훈련용 의료기기 — 정부지원금 30% 한도

- **조건**: 교육훈련용 의료기기 취득비 누계 > 정부지원금 × 30%
- **판정**: → **확인** (한도 초과 및 관리기관 승인 여부 확인)
- **확인 항목**: "교육훈련용" 명시 여부, 관리기관 사전 승인 문서
- **근거**: 운영지침 "교육훈련용 의료기기는 30% 이내로 취득 가능" [근거: 별표1 2.7, 제22조]

### SR-4: 강사료·자문료 지급 제한 인력

- **불가 유형 1**: 주관기관 소속 인력 → 강사료 지급 불가 → **확인**
- **불가 유형 2**: 사업 참여인력 → 강사료 지급 불가 (참여 여부 확인 필요) → **확인**
- **불가 유형 3**: 행정인력 → 자문료/강사료 지급 불가 (인건비로 처리해야 함) → **확인**
- **허용 유형 A**: 참여인력이 아닌 참여기관 소속 교수급 전문인력 → 결과보고서 + 자문료 지급 증빙 필요 → 있으면 **적정**
- **허용 유형 B**: 수의사·간호사·방사선사 등 전문인력이 월 4회 이하 비정기 교육 → 결과보고서 + 증빙 필요 → 있으면 **적정**
- **판정 요령**: vendorName이 기관 소속 인물인지, 참여인력 명단에 있는지 확인. 불명확하면 **확인**
- **근거**: 운영지침 별표1 "강사료 및 자문료 지급 기준" [근거: 별표1 2.2 강사료 및 자문료 지급 기준]

### SR-5: 민간부담금 포함 관리

- **원칙**: 정부지원금뿐 아니라 민간부담금의 소요비용도 사업비에 포함하여 관리
- **판정**: 민간부담금으로 집행된 건도 동일한 증빙 기준 적용 → 증빙 부재 시 **확인**
- **근거**: 운영지침 제20조 "민간부담금의 소요비용도 포함" [근거: 제20조]

### SR-6: 시설장비유지비 — 범용 장비 불인정

- **불인정 대상**: 계획서에 반영되지 않은 범용성 장비, 프린터, 복합기 등의 지출비용
- **판정**: 시설장비유지비(210-09) 집행 중 프린터·복합기·PC 등 범용 장비 → **확인**
- **근거**: 운영지침 "범용성 장비, 프린터, 복합기 등의 지출비용 집행 불가" [근거: 별표1 2.4 시설장비유지비]

### SR-7: 비목·세목간 변경 — 관리기관 승인

- **승인 필수**: 비목간 변경 → 관리기관 승인 필수
- **승인 필수**: 세목간 30% 이상 초과집행 → 관리기관 승인 필수
- **내부결재 가능**: 세목간 30% 미만 변경 → 내부 결재로 가능 (단, 증빙 제출 필요)
- **판정**: 비목간 변경·30% 이상 세목간 변경 사실이 확인되나 승인 문서 없음 → **확인**
- **근거**: 운영지침 제18조 비목변경 [근거: 제18조]

### SR-8: 회계검증수수료 — 별도 편성 필수 확인

- **원칙**: 운영비-일반수용비 내에 회계검증수수료 **필수** 편성
- **계상기준** (부가세 포함):

  |---------------------|---------|
  | 5천만원 이상 ~ 1억원 미만 | 484,000원 |
  | 1억원 이상 ~ 2억원 미만 | 545,000원 |
  | 2억원 이상 ~ 3억원 미만 | 654,000원 |
  | 3억원 이상 ~ 5억원 미만 | 800,000원 |
  | 5억원 이상 ~ 10억원 미만 | 944,000원 |

- **판정**: 집행금액이 기준금액과 불일치 시 → **확인**
- **근거**: 운영지침 별표1 일반수용비, 기타사항 회계검증수수료 [근거: 별표1 2.6, 기타사항]

### SR-9: 자문수당 한도 (시간별 3단계)

- **판정 기준**:

  | 자문 시간 | 수당 한도 |

- **주의**: 한도액은 세전(원천세 차감 전) 기준. 자문 시간 명시가 없으면 → **확인**
- **근거**: 운영지침 별표1 자문회의 지급기준 [근거: 별표1 2.2 자문회의 지급기준]

### SR-10: 여비 세부 기준

- **시내출장**:

### SR-11: 임차료 — 단순 시내이동 차량 불인정

- **불인정**: 참여인력의 단순 시내이동 출장을 위한 차량 임차
- **인정**: 교육 프로그램·행사 운영·외빈 초청 등 사업 관련 차량 임차
- **판정**: 용도 불명확 → **확인**
- **근거**: 운영지침 별표1 임차료 주석 [근거: 별표1 2.3]

## 5. 증빙 공통 규정 (제20조)

> "주관기관은 지급에 관한 **결의서 및 영수증서**와 기타 필요한 경우 **견적서 청구서 또는 계약서 검사조서** 등 지급의 내용을 증명하는데 필요한 증빙서류를 구비하여야 한다"

- 이 규정이 **전 비목 공통**으로 적용되는 유일한 증빙 기준
- e나라도움 시스템 증빙(전자세금계산서, 보조금전용카드 전표, 소득 지급명세서)은 별도 첨부 불요
- 별표2 공통: "증빙서류가 미비한 사업비 집행 금액" → 불인정

## 6. 비목별 판정 기준 (지침 명시 사항만)

> 아래는 별표1·별표2에서 **해당 비목에 대해 명시적으로 언급한 조건만** 기재.
> 비목별 구체적 증빙 목록은 지침에 없으며, 제20조 공통 규정(결의서+영수증+필요시 기타)이 적용된다.

### 6.1 인건비 (110)

- **산정공식**: 실제월지급인건비 × 참여개월 × 참여율 [근거: 별표1]
- **참여율 합계**: 100% 초과 불가 → 초과 시 **확인** [근거: 별표1]
- **포함 가능**: 기관부담 4대보험 + 퇴직급여충당금 [근거: 별표1]
- **불인정**: 개인별 계좌로 이체하지 않은 금액 [근거: 별표2]
- **불인정**: 참여연구원 외 지급, 참여율/기간 초과 [근거: 별표2]
- **불인정**: 인건비를 다른 용도로 집행한 경우 [근거: 별표2]
- **불인정**: 참여연구원 변경을 승인/통보하지 않고 인건비 지급 (참여율 변경은 자체 가능) [근거: 별표2]
- **위촉직 급여**: OP-5 적용
- **내부인건비 흡수**: DR-2 적용

### 6.2 여비 (220)

- **불인정**: 출장복명서가 없거나 출장목적이 명확하지 않은 여비 [근거: 별표2]
- **불인정**: 출장사실을 확인할 수 있는 증빙(교통비영수증 등) 미제출 [근거: 별표2]
- **불인정**: 참여연구원이 아닌 자에게 지급한 여비 (단, 외부전문가 여비 인정 가능) [근거: 별표2]
- **불인정**: 출퇴근성 교통비, 숙박 부대서비스, 여비지급규정 초과액 [근거: 별표2]
- **시내출장**: 4h미만 10,000원 / 4h이상 20,000원 [근거: 별표1]
- **시외출장**: 공무원 여비 규정 준용 [근거: 별표1]
- **국외출장/외빈초청**: 관리기관 승인 필수 [근거: 별표1]
- **출장비 식대 포함 시 회의비 식대 중복 집행**: → **확인** (SR-10)

### 6.3 자문료 / 전문가활용비 (210-01)

- **자문수당 한도**: SR-9 (시간별 3단계: 2h이하 20만/3h이하 30만/3h초과 40만) [근거: 별표1]
- **식사비**: 1인당 30,000원 이내 [근거: 별표1]
- **강사료 지급 불가**: 주관기관 소속 및 사업 참여인력 [근거: 별표1]
- **강사료 지급 인정**: 참여인력이 아닌 참여기관 소속 전문인력(교수급) → **결과보고서 및 자문료 지급 증빙 필요** [근거: 별표1]
- **강사료 지급 인정**: 전문인력(수의사·간호사·방사선사 등) 월4회 이하 비정기 교육 → **결과보고서 및 자문료 지급 증빙 필요** [근거: 별표1]
- **행정인력**: 참여인력에 포함하여 인건비 지급 (자문료 지급 불가) [근거: 별표1]
- **원천세**: DR-1 적용
- **교통비 합산**: DR-3 적용

### 6.4 회의비 (210-01)

- **불인정**: 1인당 3만원 초과 [근거: 별표2]
- **불인정**: 회의록 없이 집행 [근거: 별표2]
- **불인정**: 원격지 회의비 [근거: 별표2]
- **불인정**: 정당한 사유 없이 휴일 집행 [근거: 별표2]
- **불인정**: 주류 등 유흥성 경비, 경조사비/선물구입비 [근거: 별표2]

### 6.5 일반수용비 (210-01)

- **불인정**: 사업과 무관한 지출(기관 자체 홍보 광고료 등) [근거: 별표2]
- **불인정**: 범용성 장비(PC/주변기기/복사기/가구 등) 구입·대여·유지관리비 [근거: 별표2]
- **불인정**: 범용 소프트웨어(사무처리/백신 등) 구매 (단, 사업용 신규인력 컴퓨터 설치 SW 제외) [근거: 별표2]
- **회계검증수수료**: SR-8 기준 금액 확인 필수 [근거: 별표1]

### 6.6 재료비 (210-11)

- **부가세 포함**: DR-7 적용

### 6.7 일반용역비 (210-14)

- **2천만원 초과**: DR-8 적용 (경쟁입찰 확인)
- **부가세 포함**: DR-7 적용

### 6.8 시설장비유지비 (210-09)

- **불인정**: 사업과 관련 없는 지출, 계획서에 반영되지 않은 범용성 컴퓨터/프린터/복합기 등 (SR-6) [근거: 별표1]

### 6.9 임차료 (210-07)

- **불인정**: 참여인력의 단순 시내외 출장 차량 임차 [근거: 별표1]
- **인정**: 사업 관련 교육 프로그램·행사·외빈 초청 차량 임차 [근거: 별표1]
- **임차료 세목 50% 초과 불가** [근거: 별표1]
- **내부거래(컨소시엄간 거래, 집행기관과 거래기관 동일 등) 불가** [근거: 별표1]

### 6.10 유형자산취득비 (430)

- **전체 한도**: 정부지원금 × 40% 초과 불가 (SR-2) [근거: 별표1]
- **교육훈련용 의료기기**: 정부지원금 × 30% 이내 + 관리기관 승인 필수 (SR-3) [근거: 별표1, 제22조]
- **관리기관 승인 필수**: 사업계획서에 명시되어 있어도 집행 전 세부 구매/계약 내역 승인 필요 [근거: 별표1]
- **경쟁입찰**: 유형자산 취득은 경쟁입찰 원칙 (부득이한 경우 관리기관 승인) [근거: 별표1]
- **자산 분류 기준**: 취득가액(부가세 제외) 50만원 초과 + 사용가능년수 1년 이상 / 100만원 이상은 무조건 자산
- **50만원 초과 자산**: 자산대장 등록, **500만원 이상**: e나라도움 공시 + 센터 내 지정 장소 보관 [근거: 별표1]
- **불인정**: 기관 운영용 컴퓨터/프린터 취득 (단, 사업목적 유형자산에 부가되는 경우 계획서 반영 시 인정) [근거: 별표1]

### 6.11 업무추진비 (240)

## 7. 보완요청 문구 템플릿

### T-01: 원천세 납부증빙 요청
```
원천세 집행건에 대해 국세청 납부 여부를 확인할 수 있는 증빙(원천징수이행상황신고서 등)이 첨부되어 있지 않습니다. 해당 원천세가 실제 납부되었음을 확인할 수 있는 서류를 제출하여 주시기 바랍니다.
```

### T-02: 내부인건비 산출근거 요청
```
내부인건비 흡수 집행건에 대하여 인건비 산정기준(실제월지급인건비 × 참여개월 × 참여율)의 적정성을 확인할 수 있도록 다음 서류를 제출하여 주시기 바랍니다. 1) 사업 참여인력 명단 및 개인별 참여율 2) 참여인력의 급여대장 3) 기관부담 4대보험료 산출내역
```

### T-03: 출장복명서 요청
```
여비 집행건에 대하여 출장복명서(또는 출장결과보고서)가 첨부되지 않았거나 출장목적이 명확하지 않습니다. 출장 목적 및 결과를 확인할 수 있는 출장복명서와 출장사실을 확인할 수 있는 교통비 영수증 등을 제출하여 주시기 바랍니다. [근거: 별표2 여비 불인정]
```

### T-04: 회의비 한도 초과
```
회의비가 1인당 30,000원을 초과하여 집행되었습니다. 운영지침 별표2에 따라 회의비는 1인당 30,000원을 초과할 수 없으므로, 해당 집행의 적정성 및 참석 인원을 확인할 수 있는 회의록(방명록 포함)을 제출하여 주시기 바랍니다.
```

### T-05: 자문료 한도 초과
```
자문수당이 지급기준을 초과하여 집행된 것으로 확인됩니다. 운영지침 별표1에 따른 자문수당 한도(2h 이하: 200,000원 / 3h 이하: 300,000원 / 3h 초과: 400,000원)를 확인할 수 있도록 자문 시간 및 내용이 기재된 결과보고서를 제출하여 주시기 바랍니다.
```

### T-06: 강사료·자문료 지급 제한 인력
```
해당 집행건의 수령인이 주관기관 소속 인력 또는 사업 참여인력으로 확인될 경우 강사료 지급이 불인정됩니다. 수령인의 소속 및 참여인력 해당 여부를 확인할 수 있는 서류를 제출하여 주시기 바랍니다. [근거: 별표1 강사료 및 자문료 지급 기준]
```

### T-07: 부가세 포함 집행
```
부가가치세가 집행액에 포함되어 있습니다. 환급 가능한 매입세액이 포함된 경우 불인정 대상이므로 해당 기관의 부가세 환급 가능 여부를 확인하여 주시기 바랍니다. [근거: 별표2 공통]
```

### T-08: 유형자산취득비 한도 초과 의심
```
유형자산취득비 누계액이 정부지원금의 40%(교육훈련용 의료기기의 경우 30%)에 근접하거나 초과한 것으로 확인됩니다. 관리기관(KHIDI)의 승인 여부 및 누계 비율을 확인할 수 있는 서류를 제출하여 주시기 바랍니다.
```

### T-09: 시설장비유지비 범용장비 집행
```
시설장비유지비에 프린터, 복합기 등 범용성 장비 관련 비용이 집행된 것으로 확인됩니다. 운영지침에 따라 범용성 장비의 집행비용은 인정되지 않으므로, 해당 집행의 사업 전용성 여부를 확인할 수 있는 서류를 제출하여 주시기 바랍니다.
```

### T-10: 비목·세목 변경 승인 확인
```
비목간 변경 또는 세목간 30% 이상 초과집행이 확인됩니다. 운영지침 제18조에 따라 관리기관(KHIDI) 승인을 받은 서류를 제출하여 주시기 바랍니다.
```

### T-11: 교통비 합산 집행 (안내)
```
교통비영수증(KTX 등) 및 지출발의서상 자문수당과 교통비가 합산 집행된 것을 확인하였습니다. 외부전문가 교통비는 여비 비목에서 별도 집행하는 것이 원칙이므로, 전문가활용비에 교통비를 합산 집행한 사유를 확인하여 주시기 바랍니다.
```
