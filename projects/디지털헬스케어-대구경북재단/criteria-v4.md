## 2. 인건비 (110)

### 2-1. 보수(01) -- 정규직원 전담인력 인건비
### 2-2. 상용임금(03) -- 무기계약직
### 2-3. 일용임금(04) -- 수개월/수일 일용, 기간제 근로자 보수

### 계상 범위 [근거: 세칙 별표1]:
- 근로계약서에 따라 지급하는 급여 및 상여금 총액
- 기관에서 공통적으로 지급하는 수당 등
- 퇴직금 지급을 위한 퇴직충당금의 적립액
- **기관부담 4대보험 및 퇴직급여충당금 포함** [근거: 세칙 별표1 인건비 주석 -- 빨간색 강조]
- **인건비 = 실제월지급인건비 x 참여개월 x 참여율** (참여율이란 참여인력의 연봉총액을 100으로 할 때 지원사업에 참여하는 인력에게 지급하는 인건비의 비율)
- 인건비는 주관기관과 참여기관을 포함하여 **자기부담금을 제외한** 총 사업비(국비+지방비)의 **30% 이하**로 계상 [근거: 세칙 별표1 인건비 주석]
  - **예외**: 불가피하게 인건비 예산 기준이 총 사업비(국비+지방비)의 30%를 초과하는 경우, **사유서 및 공문을 제출하여 전담기관의 승인**을 득하여야 함 [근거: 세칙 별표1 인건비 주석]
- **기준단가 참조**: 인건비는 협약 당해연도 **기획재정부 계약예규 학술연구용역인건비 기준 단가**를 참조하되, 참여기관의 자체 인건비 산정 기준을 준용하여 집행할 수 있음 [근거: 세칙 별표1 인건비 주석]

### 필수 증빙:
- 지출결의서
- 급여내역/급여명세서
- 개인별 계좌이체 확인 (통장사본 또는 이체내역)

### 불인정 기준 [근거: 세칙 별표2 인건비]:
| # | 불인정 사유 |
|---|---------|
| L-1 | 참여연구원 변경을 **전담기관**에 승인 또는 통보하지 않고 인건비가 지급된 경우 |
| L-2 | 참여연구원 이외 지원인력의 급여를 인건비로 집행한 경우 |
| L-3 | 개인 참여율 및 참여기간을 초과하여 집행한 경우 |
| L-4 | **개인별 계좌로 이체하지 않은 금액** |
| L-5 | 인건비를 다른 용도로 집행한 경우 |

### 정산검토 시 체크포인트:
- [ ] 지출결의서 첨부 여부
- [ ] 급여내역/급여명세서 첨부 여부
- [ ] 위촉직/기간제: 근로계약서 확인
- [ ] 내부인건비 흡수: 참여율 확인서류 존재 여부 -> 없으면 **확인**
- [ ] 인건비 단가가 기획재정부 학술연구용역인건비 기준단가 대비 현저히 높은 경우 -> **확인**

## 3. 여비 (220)

### 3-1. 국내여비 (01) -- 국내 출장경비, 각 기관이 정한 기준에 따른 실 소요 경비

### 필수 증빙:
- 출장신청서 (근무지외출장/근무지내출장)

- 지출결의서
- 교통비 영수증 (KTX, 택시, 하이패스 등)

### 불인정 기준 [근거: 세칙 별표2 여비]:
| # | 불인정 사유 |
|---|---------|
| T-1 | 과제와 관련 없는 개인 목적으로 국내외 여비가 지급된 경우 |
| T-2 | 참여연구원이 아닌 인원에게 지출된 여비 (단, 사업 관련 외부전문가 여비 인정 가능) |
| T-3 | 출장(출장비에 식대가 포함된 경우) 중 회의비(식대)를 사용한 경우 해당 식대 |
| T-4 | 국내외여비에 포함되는 숙박비, 식비 등을 출장지 관계기관에서 지급한 경우 |
| T-5 | **출장복명서가 없거나 출장목적이 명확하지 않은 여비** |
| T-6 | 출장기간 초과, 체재지역 외 사용액 |
| T-7 | 여비지급규정 초과액 |
| T-8 | **국외여비 지급일 또는 국외출장 결재일의 매매기준율을 적용하지 않은 경우 그 차액** |

> ※ 출장 여비는 **참여기관의 여비지급 기준**에 따른다. 기준이 마련되어 있지 않은 경우는 **공무원 여비규정**을 따른다. [근거: 세칙 별표2 여비 각주]

### 3-2. 국외여비 (02)
- **전담기관(관리기관)의 승인**을 득하여야 함 [근거: 세칙 별표1 국외여비 주석]
- 불인정 사례 [근거: 세칙 별표2 여비]:
  - 참여연구원이 아닌 자에게 지급한 여비 (단, 사업수행을 위한 교육, 세미나 등에 초청한 외부 전문가에게 지급하는 교통비 지급은 인정)
  - 참여연구원의 출/퇴근성 교통비를 지급한 경우
  - 숙박비 중 부대서비스 이용 비용
  - 자가용 출장 시 동승자에게 운임을 지급한 경우
  - 항공기 내 숙박을 여비로 지급한 경우
  - **지급일 또는 결재일의 매매기준율 미적용 시 그 차액** [근거: 세칙 별표2 여비]

### 정산검토 시 체크포인트:
- [ ] 출장신청서(근무지외출장) 첨부 여부
- [ ] **출장복명서 첨부 여부** -> 없으면 **확인** (T-5) -- 금액/출장유형 무관, 모든 출장에 필수
- [ ] 지출결의서 첨부 여부
- [ ] 교통비 영수증 첨부 여부 (KTX운임, Receipt, 하이패스, IMG/JPG 사진 포함)
- [ ] 출장자 이름이 출장신청서와 일치하는지
- [ ] 여비지급규정 초과 여부 (금액 비교)
- [ ] 국외여비: 매매기준율 적용 여부 확인

#### 회의비 증빙서류 [근거: 세칙 별표1]:
- **내부결재문서** (회의목적, 일시, 장소, 대상, 산출내역 포함)
- **회의록(또는 회의결과보고서) 및 방명록(서명포함)**
- 주말/공휴일 집행 시 근무 또는 사업수행을 확인할 수 있는 서류 및 증빙자료
- 쇼핑몰에서 사용한 경우 거래 내역 확인 가능한 증빙(영수증) 첨부

#### 회의비 주의사항 [근거: 세칙 별표1]:
- 동일 기관 사업추진비 집행시 **과제 미참여인력 참석 필수**
- 참석자 서명이 있는 회의록에 소속 성명 기재, **과제 참여/미참여 인력 구분** 가능해야

### 계약 관련 규정 [근거: 세칙 별표1]:
- 협약 시 사업계획서에 명시한 용역 외 사업 수행 과정에서 추가로 용역 계약을 하고자 하는 경우 **전담기관의 승인(공문)을 득해야 함**
- 「국고보조금 통합관리지침」에 따라, **2천만원(공급가액 기준)을 초과하는 물품 및 용역구매 계약을 체결하는 경우에는 원칙적으로 일반경쟁에 부쳐야 함**

  - (3) **집행금액의 적정성** 등을 확인한 후 계약 체결
- **수의계약 시 내부평가 문서 필수**: 해당 업체를 최종적으로 선정한 **내부평가 문서**가 존재해야 함 [근거: 세칙 별표1 일반용역비 비고란]
- 경쟁입찰 대상임에도 수의계약을 진행할 경우, **동일한 조건을 가진** 최소 2개 이상의 다른 업체로부터 추가 견적을 받아 적정성 등을 평가한 내부문서를 **증빙자료로서 구비** [근거: 세칙 별표1]
  - ※ "동일한 조건을 가진 업체"란 최종 업체 선정 시 참여기관에서 제시한 수의계약 사유와 동일한 조건을 가진 업체를 의미 (예: 수의계약 사유가 '여성기업지원에 관한 법률' 제2조제1호에 따른 여성기업인 경우, 비교 견적을 받는 다른 업체들도 여성기업에 해당되어야 함) [근거: 세칙 별표1 빨간색 강조]

- 50만원 초과 물품은 **자산대장에 등록**, 500만원 이상은 **국고보조금통합관리시스템에 공시**, 센터 내의 지정된 장소에서 보관 관리 [근거: 세칙 별표2 유의사항]

## 14. 판정 기준 요약

### "적정" (증빙 충분):
- 해당 비목의 필수 증빙이 모두 첨부되어 있고
- 불인정 사유에 해당하지 않으며
- 금액이 한도 이내인 경우

### "확인" (추가 검토 필요):
- 필수 증빙 중 하나 이상 미첨부
- 불인정 사유에 해당할 가능성이 있는 경우
- 금액이 한도를 초과하거나 초과 여부를 확인할 수 없는 경우
- 사업 관련성이 불분명한 경우

### "SKIP":
- totalAmount = 0 (취소전표)
