## 1. 수치 정확성 오류

**[1-1] 연구수당 한도 산식 명칭 오류**
- 섹션: `4. 연구수당`
- 문제: criteria가 "수정인건비 = 해당 과제 인건비 + 학생인건비 + 미지급인건비"로 정의하지만, 고시 제26조①의 "수정인건비" 정의와 정확히 일치하는지 확인 필요. "수정직접비"(직접비에서 현물부담액·위탁·국제공동·부담비 제외)와 혼용 위험
- 수정 제안: 고시 제26조① 원문과 대조하여 정확한 산식 명칭으로 수정

**[1-2] 학생인건비 하한선 적용 대상 기관 범위 누락**
- 섹션: `1-5. 학생인건비`
- 문제: "정부출연기관·대학 기준"으로만 표기. 기타비영리기관의 학생인건비 하한 적용 여부(제57조 이하) 미명시
- 수정 제안: 기관 유형별 하한선 적용 여부를 명시

## 2. 원천징수영수증 단독 인정 문제

**[2-1] 원천징수영수증 = 월별 급여 확인 불가**
- 섹션: `0. 판정 원칙`, `1-1`, `1-3`
- 문제: "원천징수영수증 = 급여 지급 사실 확인 가능"은 잘못된 설명. 원천징수영수증은 연간 총급여 합계만 표시하며, 월별 지급액·지급일 확인 불가
- 고시 근거: 제22조② — 사용 입증자료 구비 의무
- 수정 제안: "원천징수영수증은 연간 총급여 수준 확인에만 사용 가능. 월별 인건비 계상률 검증 및 지급 사실 확인은 급여명세서(월별) + 계좌이체증명이 필수"

## 3. 회의비 2024.12.1 기준 법적 근거

**[3-1] 고시 부칙에 해당 경과규정 없음**
- 섹션: `3-3. 회의비`
- 문제: 2024.12.1 기준은 현행 고시(제2025-9호, 2025.3.6. 시행) 본문 및 부칙에 직접 명시 없음. 이전 개정 고시(제2024-xx호) 부칙 경과규정 또는 전문기관 운영지침에서 유래한 것으로 추정
- 수정 제안: 해당 개정 고시 번호와 부칙 조항으로 명시하거나, "전문기관 운영지침에 따름"으로 표기. 현행 고시 제25조④가 근거라면 시행일(2025.3.6.) 기준으로 단순화 가능

## 4. 누락된 증빙 요건

**[4-1] 외부인건비 — 4대보험 기관부담분 처리 기준 누락**
- 섹션: `1-3. 외부인건비`
- 문제: 외부 근로자 4대보험 기관부담금 계상 시 증빙(4대보험 납부확인서, 산출근거) 누락
- 고시 근거: 제6조3호 — 참여연구자 4대보험 기관부담금도 인건비 사용용도에 포함

**[4-2] 연구수당 — 지급 시기 요건 누락**
- 섹션: `4. 연구수당`
- 문제: 고시 제26조③에 따라 연구수당은 "단계 종료 후 1개월 이내" 지급 의무. 기한 요건 미명시 시 기한 초과 집행을 점검완료 처리할 위험
- 수정 제안: "단계 종료 후 1개월 이내 미지급 시 불인정" 항목 추가

**[4-3] 위탁연구개발비 — 위탁기관 정산서 누락**
- 섹션: `6. 위탁연구개발비`
- 문제: 계좌이체증명만 필수 증빙. 위탁기관이 제출한 정산서(집행내역)를 주관기관이 보유·확인해야 함
- 고시 근거: 제27조⑤ — 위탁기관 정산 결과 확인 의무
- 수정 제안: 위탁기관 정산서 미첨부 케이스 보완요청 문구 추가

## 5. 비목 분류 오류

**[5-1] 소프트웨어 활용비 비목 독립 처리 필요**
- 섹션: `3-9. 그밖의비용`에 통합됨
- 문제: 고시 제10조5호는 소프트웨어 활용비를 독립 비목으로 열거. 연구기간 초과 라이선스의 최소단위 소명 등 별도 집행 조건 존재
- 수정 제안: 소프트웨어 활용비를 3-10으로 독립하여 별도 판정 기준 수립

**[5-2] 지식재산 창출 활동비 비목 누락**
- 섹션: 전체
- 고시 근거: 제10조1호 — 특허·표준 정보 조사·분석 비용 등
- 문제: 특허출원비는 간접비-성과활용지원비, 특허 사전 정보조사비는 지식재산 창출 활동비(직접비 가능). 두 비목 경계 기준 없어 혼동 위험
- 수정 제안: 지식재산 창출 활동비 항목 추가하여 경계 명확화

## 6. 실무적으로 문제될 보완 문구

**[6-1] 검수확인서 사진 조항 오인용** (기존 지적 재확인)
- "국가연구개발혁신법 사용기준 제22조 3항" → "고시 제23조 및 시설장비관리 표준지침"으로 수정 필요

**[6-2] 퇴직급여충당금 근거 서술 부정확**
- 섹션: `1-1. 내부인건비(비영리)` 보완요청 문구
- 문제: "추후 1년이상 근속을 확인가능한 증빙"은 실무상 이행 불가능. 정산 시점에서 미래 근속 여부를 증빙할 수 없음
- 수정 제안: "재직 중인 경우 근속기간 확인 자료(재직증명서 등)"로 수정

**[6-3] 영리기관 연구실운영비 별지4호 조건 불명확**
- 섹션: `3-7. 연구실운영비`
- 문제: "연구개발계획서에 첨부한 경우에만 계상가능"이라 했으나, 고시 제68조④는 별지4호를 첨부하여 **중앙행정기관의 승인**을 받는 것이 조건. 단순 첨부가 아닌 승인 여부 확인이 핵심
- 수정 제안: "중앙행정기관의 승인 여부 확인"을 명시
