# 이지바로 상시점검 판정 기준 (혁신법 공통)

> **법적 근거**: 국가연구개발혁신법, 국가연구개발사업 연구개발비 사용 기준(과기부고시 제2025-9호, 2025.3.6. 시행)
> **작성일**: 2026-02-18
> **버전**: v1 (초안 — 4모델 교차검증 전)
> **적용 범위**: 통합이지바로(EZBaro) 상시점검 대상 모든 국가R&D 과제

---

## 0. 판정 원칙

1. **문서 내용 종합 판단** — 파일명이 아닌 OCR 내용으로 문서 종류를 판단한다.
   - "품의서" = 내부결재문서로 인정
   - "지출결의서" = 내부결재문서로 인정
   - "원천징수영수증" = 급여 지급 사실 확인 가능
2. **보수적 판단** — 확실한 경우만 "점검완료", 나머지는 "보완요청"
3. **연구비카드 또는 계좌이체 원칙** (고시 제22조⑤) — 현금 사용 시 사유 소명 요구
4. **부가세 환급 가능분 불인정** (고시 제21조④1) — 영리기관에 특히 적용
5. **사업기간·기관유형·비목에 따라 판단** — 영리/비영리 기준이 다름

## 1. 인건비

### 1-1. 내부인건비 (비영리기관: 대학, 정부출연기관, 기타비영리)

**법적 근거**: 고시 제6조, 제39조(정부출연기관), 제48조(대학), 제57조(기타비영리)

**적정 조건** (모두 충족 시 점검완료):
- ① 참여연구자현황표 (연구자명, 참여기간, 인건비계상률 또는 미지급인건비계상률, 변경사항)
- ② 급여명세서(월별) **또는** 원천징수영수증 (총급여 확인 가능)
- ③ 계좌이체증명 (급여 실지급 확인)

**불인정 조건**:
- 미참여 연구원 인건비 → 환원 대상
- 집행금액이 계상률×급여 한도 초과

**보완요청 문구**:

| 누락 상황 | 보완요청 문구 |
|----------|-------------|
| 전부 없음 | 1. 월별 총급여 및 지급사실을 확인 가능한 증빙([원천징수영수증] 또는 [급여명세서&계좌이체증명]) 2. 참여연구자현황표(연구자명, 참여기간, 인건비계상률 또는 미지급인건비계상률, 변경사항 등) 보완 바랍니다. |
| 현황표만 없음 | 참여연구자현황표(연구자명, 참여기간, 인건비계상률 또는 미지급인건비계상률, 변경사항 등) 보완 바랍니다. |
| 급여명세서만 없음 | 참여연구원의 월별 총급여를 확인 가능한 증빙(급여명세서, 원천징수영수증 등) 보완 바랍니다. |
| 계좌이체증명만 없음 | 계좌이체증명 보완 바랍니다. |
| 현황표+급여명세서 없음 | 1. 참여연구자현황표(연구자명, 참여기간, 인건비계상률 또는 미지급인건비계상률, 변경사항 등) 2. 참여연구원의 월별 총급여를 확인 가능한 증빙(급여명세서, 원천징수영수증 등) 보완 바랍니다. |
| 현황표+계좌이체 없음 | 1. 참여연구자현황표(연구자명, 참여기간, 인건비계상률 또는 미지급인건비계상률, 변경사항 등) 2. 계좌이체증명 등 보완 바랍니다. |
| 급여명세서+계좌이체 없음 | 1. 참여연구원의 월별 총급여를 확인 가능한 증빙(급여명세서, 원천징수영수증 등) 2. 계좌이체증명 등 보완 바랍니다. |
| 퇴직급여충당금 근거 미비 | 퇴직급여 적립에 대한 근거(현재 또는 추후 1년이상 근속을 확인가능한 증빙, 또는 1년 미만 근속자에 대한 퇴직금 지급 규정 등) 보완바랍니다. |
| 4대보험 기관부담금 | 1. 내부결재문서 2. 계좌이체증명 3. 4대보험 사용자부담금에 대한 산출 근거 자료 등 보완바랍니다. |
| 한도 초과 | 집행금액이 한도(=급여액 × 계상률)를 초과하는 것으로 확인 됩니다. 과제기간 종료 전 재검토 후 계상률 변경 등 조치 바랍니다. |
| 미참여 연구원 | 미참여 연구원에 대한 인건비는 불인정 대상입니다. 집행 취소 바랍니다. |
| 현물인건비 확인서 | 국가연구개발사업 연구개발비 사용기준의 별지서식 6호인 현물부담확인서를 보완 바랍니다. |

### 1-2. 내부인건비 (영리기관)

**법적 근거**: 고시 제65조, 제6조

**적정 조건** (모두 충족 시 점검완료):
- ① 참여연구자현황표 (현금/현물 구분 명시)
- ② 급여명세서 또는 근로자별 근로소득원천징수 확인서
- ③ 계좌이체증명
- ④ 건강보험자격득실확인서 (신규채용 참여연구자)
- ⑤ 별지 제3호서식 — 영리기관 현금계상 인건비 관리계획 및 현황

**보완요청 문구**:

| 누락 상황 | 보완요청 문구 |
|----------|-------------|
| 전부 없음 | 1. 참여연구자현황표(참여연구자별 현금/현물 구분, 참여기간, 인건비계상률 표기) 2. 급여명세서(또는 근로자별 근로소득원천징수 확인서) 3. 계좌이체증명 4. 건강보험자격득실확인서(영리기관 신규채용 참여연구자에 한함) 5. 영리기관의 현금 계상 인건비 관리 계획 및 현황(연구개발비 사용 기준 [별지 제3호서식]) 등 보완 바랍니다. |
| 신규채용 확인 미비 | 1. 신규채용인원임을 확인하기 위한 건강보험자격득실확인서를 보완 바랍니다. 2. 영리기관의 현금 계상 인건비 관리 계획 및 현황(연구개발비 사용 기준 [별지 제3호서식]) 제출 바랍니다. |
| 기존인력 현금인건비 | 1. 기존인력인 경우 현금인건비를 받는 경우 현금인건비 지급요건을 확인할 수 있는 서류 제출 바랍니다. 2. 영리기관의 현금 계상 인건비 관리 계획 및 현황(연구개발비 사용 기준 [별지 제3호서식]) 제출 바랍니다. |

### 1-3. 외부인건비

**법적 근거**: 고시 제39조⑩, 제48조⑧, 제65조⑨

**적정 조건**:
- ① 근로계약서 (또는 재학증명서 — 타 대학 소속 학생)
- ② 건강보험자격득실확인서
- ③ 과제참여계약서 (프리랜서·개인사업자)
- ④ 외부참여연구자 소속기관장 확인서 (해당 시)
- ⑤ 급여명세서(월별) + 계좌이체증명 또는 원천징수영수증

**보완요청 문구**:

| 누락 상황 | 보완요청 문구 |
|----------|-------------|
| 전부 없음 | 1. 근로계약서(타 대학 소속 학생연구자의 경우 재학증명서) 2. 건강보험자격득실확인서 3. 프리랜서 및 개인사업자의 경우 과제참여계약서 4. 해당되는 경우 외부참여연구자 소속 기관장 확인서 5. 월별 총급여 및 지급사실을 확인 가능한 증빙([원천징수영수증] 또는 [급여명세서&계좌이체증명]) 6. 참여연구자현황표 등 보완 바랍니다. |
| 급여명세서+계좌이체 없음 | 1. 급여명세서(월별) 2. 월 급여 총액에 대한 계좌이체증명 보완 바랍니다. |

### 1-5. 학생인건비

**법적 근거**: 고시 제7조, 제40조, 제49조

**계상 하한** (정부출연기관·대학 기준):
- 학사과정: 월 130만원 이상

| 누락 상황 | 보완요청 문구 |
|----------|-------------|
| 비통합 - 증빙 없음 | 1. 참여연구자현황표(연구자명, 해당 기관의 학생인건비 계상기준, 참여 기간, 학생인건비계상률, 변경사항 등) 2. 계좌이체 증명 3. 학생연구자 연구참여확약서(재학) 또는 근로계약서(휴학) 보완바랍니다. |

## 2. 연구장비재료비

### 2-1. 연구시설장비비

**법적 근거**: 고시 제8조, 제23조, 제73조

**적정 조건** (모두 충족):
- ① 내부결재문서 (해당 과제와의 직접 관련성 포함)
- ② 구매의뢰서
- ③ 거래명세서
- ④ **검수(설치)완료확인서** — 검수자, 검수일자, 서명, **사진 필수** (고시 제23조)
- ⑤ 계약서 (해당 시)

**주요 한도**:
- **3천만원 이상**: ZEUS(연구시설장비종합정보시스템) 등록 필수 (고시 제23조④)
- **3천만원 이상 신규·변경 구입**: 중앙행정기관 사전승인 필요 (고시 제73조①5)
- **구입완료 시한**: 과제 종료일 **2개월 전** (고시 제23조⑤4)

| 누락 상황 | 보완요청 문구 |
|----------|-------------|
| 카드전표만 있음 | 1. 내부결재문서(해당 과제와의 직접적인 관련성을 포함하여 작성) 2. 거래명세서 3. 검수(설치)완료확인서(사진첨부) 4. 계약서(해당시) 등 보완 바랍니다. |
| 검수확인서 누락 | 검수(설치)완료확인서(검수자, 검수일자, 검수자의 서명, 시설·장비의 사진 포함) 보완 바랍니다. |
| 검수확인서 사진 누락 | 국가연구개발혁신법 사용기준 제22조 3항에 따르면 연구시설장비비에 대한 필수증빙인 검수(설치)완료확인서에는 사진첨부가 필수적입니다. 보완바랍니다. |
| 3천만원 이상 ZEUS 미등록 | 관리지침상 부가가치세 포함 3천만원 이상 기자재를 구입하는 경우 ZEUS(연구시설장비종합정보시스템)에 등록을 하도록 되어 있으며 등록증 미제출시 불인정 처리 되오니 등록증을 보완바랍니다. |
| 3천만원 이상 미승인 구입 | 관리지침상 3천만원 이상 기자재를 신규구입하거나 변경구입하는 경우 전문/관리기관에 사전 승인을 받도록 되어 있으며, 미승인분은 불인정 처리 대상입니다. |
| 외자구입 서류 미비 | 외자구매(국외 수입)일 경우 수입신고 서류. 단, 관세법상 수입신고가 필요없는 물품인 경우에는 배달증명 자료(운송장 등)를 보완 바랍니다. |
| 종료 2개월전 구입 | 사업기간 종료 2개월전까지 구입 및 입고가 완료되지 않은 연구시설장비비 집행액은 불인정 처리 대상이므로 연구비 환원처리 하시기 바랍니다. |

## 4. 연구수당

**법적 근거**: 고시 제11조, 제26조

**1인 한도** (고시 제26조⑥):

**적정 조건** (모두 충족):
- ① 내부결재문서

| 기여도평가서류 없음 | 연구책임자의 참여연구자에 대한 기여도 평가서류(평가기준, 방법이 포함된 평가계획 및 결과 등) 보완바랍니다. |
| 기여도평가 대상자 누락 | 기여도 평가는 해당 단계에 참여한 모든 연구원(중도입사, 퇴사자 포함)을 대상으로 하여야 합니다. 기여도 평가서류 보완 바랍니다. |
| 개인별 70% 초과 | 참여연구원 1인에 대한 연구수당 지급 비율이 70퍼센트를 초과하는 경우 초과분은 불인정 대상입니다. |

| 연구비카드 미사용 | 연구개발비는 연구비카드 또는 거래처에 대한 계좌이체 형태로 사용함을 원칙으로 합니다. (고시 제22조⑤) 해당 규정 미준수에 대한 사유를 소명 바랍니다. |
| 과제 연관성 소명 | 해당 과제와의 직접적인 연관성을 소명바랍니다. 또한 연구개발계획서 업로드 바랍니다. |
| 간접비성 경비 직접비 집행 | 전기료, 통신료, 관리비 등은 간접비로 사용하는 것이 타당합니다. 각 개별과제에 소요된 내역에 대해 합리적으로 산출이 가능한 자료가 있는 경우 업로드 요청드리고, 그렇지 않은 경우 환원 처리 바랍니다. |
| 특허출원비 비목 오류 | 특허출원비 등 지식재산권 출원 등록에 관한 비용은 '간접비-성과활용지원비'로 집행해야 합니다. 비목 변경 바랍니다. (고시 제17조②) |

## 9. AI 판정 주의사항

1. **문서 유형 종합 판단**
   - "품의서", "지출결의서", "구매요청서" → 내부결재문서로 인정
   - "원천징수영수증" → 급여 지급 사실 확인 가능 (급여명세서 대체 가능)
   - 파일명만 보지 말고 OCR 내용으로 판단

2. **영리/비영리 구분 필수**
   - 영리기관: 인건비 현물원칙, 연구실운영비 소모성비용 불가, 부가세 환원
   - 비영리: 교원 인건비 현물원칙

3. **기간 관련 주의**
   - 2024.12.1 이후 회의비 식대: 사전내부결재 필수
   - 종료 2개월 전: 연구시설장비비 구입완료 필요
   - 종료 1개월 전 다량 구매: 소명 요구

4. **불인정 vs 보완요청 구분**
   - 명백한 불인정: 미참여 연구원 인건비, 외부인원 없는 회의비 식대, 소모성비용(영리 연구실운영비)
   - 증빙 보완 가능: 나머지 대부분
