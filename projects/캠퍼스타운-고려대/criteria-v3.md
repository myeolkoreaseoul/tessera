## I. 공통 규정

### 1. 집행 원칙 [근거: 지침 제4조, 제16~17조]
- 「지방자치단체 보조금 관리에 관한 법률」, 「지방재정법」, 「서울시 지방보조금 운영 관리지침」 준수
- 인건비성 경비 제외한 모든 비목: **보조금결제 전용카드** 사용 원칙 [근거: 제16조②]
- 지출거래 시 **전자세금계산서** 사용 원칙 [근거: 제16조②]
- 소상공인 간편결제 사용 시 서울시 사전 협의 필요 [근거: 제16조②]
- 현금 인출 불가; 부득이한 경우 서울시 사전 승인 필요 [근거: 제16조④]
- 사업비는 회계연도 말(매년 12.31)까지 집행, 이월 불가 [근거: 제16조⑤]
- 지출원인행위(계약, 물품구매 등) 없이 임의 인출 후 사후정산 엄격 금지 [근거: 제16조③]

### 2. 계약 규정 [근거: 지침 제17조]
- 추정가격 **2천만원 초과** 공사·용역·물품계약: 나라장터(국가종합전자조달시스템) 이용 필수
- 추정가격 **2천만원 이하** 공사·용역·물품(수의계약 포함)도 나라장터 이용
- 수의계약 체결 시 **동일업체와 연 4회 초과 금지** (2025.7 신설) [근거: 제17조⑦]
  - 확인 방법: 해당 업체와의 당해 연도 계약 체결 현황표 검토
- 추정가격 1천만원 이하 물품 임차는 일반지출로 집행 가능

### 3. 사업비 집행 우선순위 [근거: 지침 제17조①]
- **대학 대응자금을 보조금(시비)보다 우선 집행**해야 함
- 정산 시 대응자금 집행 현황 확인 필수; 대응자금 미집행·부족 집행 시 보조금 반환 조치 가능

### 4. 사업계획 변경 제한 [근거: 지침 제17조②]
- 사업계획 변경: **연 4회**까지 가능 (11월 30일까지 서울시 사전 승인)
- AI 인재양성 프로그램 추가 시에 한해 **연 5회**까지 가능
- 미승인 사업계획 변경 후 집행한 경우 → **보조사업비 지출로 불인정**

### 3. 인건비성 경비 원천징수 [근거: 지침 제18조]
- 강사비, 회의비, 자문비, 심사평가비, 멘토비, 단순인건비 등 기타소득이 **건당 125,000원 초과** 시 원천징수 의무
- 원천징수 후 관할 세무서에 신고·납부 (익월 10일까지)
- 원천징수영수증 첨부 필수

### 4. 물품 관리 [근거: 지침 제19조]
- 캠퍼스타운 사업 관련 물품·장비 확보는 **임차(리스) 원칙**
- 단가 50만원 이상 물품은 중요재산 관리 대상 (e나라도움 등록)
- 인테리어 비용: 집행 불가 [근거: 운영지침 변경 2025.7]

### 5. 시외여비 사전 승인 (2025.7 신설) [근거: II.집행기준 여비]
- 시외여비 발생 출장 시 **서울시 및 자치구 사전 승인 문서** 제출 의무화 (2025.7 신설) [근거: 증빙서류 p.26]

### 6. 이행보증보험 [근거: 지침 제10조③]
- 대학은 협약 기준일(2025.1.1.)로부터 **2개월 이내** 자부담으로 이행보증보험 계약 체결·보험증권 제출

### 7. 임차시설 원상복구비 집행 불가 [근거: 지침 제19조⑦]
- 임차시설 만료 후 발생하는 **원상복구 비용은 사업예산으로 집행 불가**
- 임대차계약서에 "임차인이 임차물을 창업공간으로 활용하기 위해 공사 시행하되 원상복구 의무 면제" 특약 설정 의무

### 8. 100만원 이상 거래 시 비교견적서 의무 [근거: 지침 붙임6]
- **100만원 이상** 거래 시 견적서와 **비교견적서** 모두 첨부 의무 (지출 근거로 첨부)
- 각 비목별 체크리스트의 "견적서(비교견적서)" 항목은 이 기준 적용

### 9. 사업계획 변경 승인 없는 임의 집행 환수 [근거: 지침 제17조⑤, 제23조②]
- 서울시 사전 승인 없이 임의로 사업 변경 후 보조금 집행 시 해당 금액 **환수 가능**
- 보조금 지출이 당해 보조사업과 무관한 경우 → 보조사업비 지출액으로 **불인정**

### 10. 대응자금 집행률 저조 시 비례 환수 [근거: 지침 제23조③]
- 대응자금 집행률이 낮을 경우 총 집행액 기준으로 **보조금:대응자금 비율로 나누어 정산** 후 보조금 반환 조치 가능
- 정산 검토 시 대응자금 집행 현황 확인 필수

### 11. 부가가치세 환급금 처리 [근거: 지침 제24조⑥]
- 사업 수행 중 발생한 **부가가치세 환급금은 집행금액에서 공제 또는 반납** 원칙
- 재투자 시 서울시 사전 승인 필요

#### 2-1. 강사비 [e나라도움: 기타인건비(110-05)]
**개념**: 본 사업 관련 강좌 개설 및 강의를 맡은 자 및 보조하는 자에 대한 수당

**집행 기준**:
- 서울시 인재개발원 강사료 및 원고료 지급기준 준용, 등급에 따른 지급범위 내에서 지급 가능
- 비대면 교육: 실시간으로 화상교육을 실시한 경우에 한하여 동일 기준 적용 (녹화 강의 불인정)

**적정 조건**:
- [ ] 내부결재(계획·결과보고) 확인
- [ ] 이력서 첨부 (최초 집행시 1회)
- [ ] 강의확인서(서명) 확인
- [ ] 강의증빙자료(교안, 강의사진, 출석부 중 1개) 확인
- [ ] 계좌이체확인 확인
- [ ] 기타소득 125,000원 초과 시 원천징수영수증 첨부

**확인 사유**:
- 이력서 미첨부 (최초 집행 여부 확인 필요)
- 강의확인서 미서명
- 강의증빙자료(교안/사진/출석부) 전혀 없음
- 비대면 강의임에도 실시간 화상 여부 불명확
- 지급액이 서울시 인재개발원 기준 초과 의심

- 캠퍼스타운 전담조직 소속 인력에게는 **지급 불가**

**적정 조건**:
- [ ] 내부결재(계획·결과보고) 확인
- [ ] 참석자명부(서명) 확인
- [ ] 심사 또는 평가결과서 확인
- [ ] 이력서 첨부 (최초 1회)
- [ ] 계좌이체확인
- [ ] 단가 기준 이내 확인 (시간당 10만원, 1일 50만원)
- [ ] 서면 심사의 경우 7만원/일 이내 확인
- [ ] 전담조직 소속 여부 확인

**확인 사유**:
- 시간당 10만원, 1일 50만원 초과
- 서면 심사임에도 7만원/일 초과
- 전담조직 소속 인력에게 지급
- 참석자명부 미서명
- 심사·평가 결과서 없음

#### 2-3. 멘토비 [e나라도움: 기타인건비(110-05)]
**개념**: 창업기업이 창업활동과 관련, 전문지식이 필요하여 대학 내·외부 전문가로부터 자문(멘토링)을 받는데 소요되는 경비

- **1인 멘토에게 월 200만원 초과 지급 불가**

**적정 조건**:
- [ ] 멘토링내역서(서명) 확인
- [ ] 지급내역서 확인
- [ ] 계좌이체확인
- [ ] 이력서·경력증명서 (최초 1회)
- [ ] 1일 30만원(대면)/15만원(비대면) 이내 확인
- [ ] 동일 멘토 월 누계 200만원 이하 확인
- [ ] 기타소득 125,000원 초과 시 원천징수영수증 첨부

**확인 사유**:
- 1일 단가 초과 (대면 30만원, 비대면 15만원)

- 멘토링내역서 미서명

- [ ] 회의록 확인

- [ ] 기타소득 125,000원 초과 시 원천징수영수증 첨부

**확인 사유**:
- 대면 2h 기준 15만원/20만원 초과
- 비대면 2h 기준 5만원/7만원 초과
- 전담조직 소속 인력에게 지급
- 회의록 미첨부
- 참석확인서 미서명

#### 2-6. 프로그램용역비 [e나라도움: 일반용역비(210-14)]
**개념**: 프로그램 운영과정에서 사업단 내에서 자체적으로 추진할 수 없는 전문분야의 업무를 용역을 통해 대행시키는 비용

**집행 기준**:
- 외주용역 시 사업단의 업무 범위 vs 외주용역의 업무 범위 **명확히 기재** 필수
- **일괄용역 불가** (사업단 업무 전체를 외부에 맡기는 형태)
- 사업비목으로 편성된 항목(회의비, 자문비, 식비, 다과비 등)을 프로그램용역비 내에 **중복 편성 불가**
- 추정가격과 내용에 따라 **지방계약법 준수** (나라장터 이용 등)

**적정 조건**:
- [ ] 프로그램기획서 확인

## III. results.json 작성 형식

```json
{
  "rowNum": 1,
  "type": "비목명",
  "purpose": "집행용도",
  "amount": 금액,
  "vendor": "거래처",
  "date": "집행일",
  "status": "적정 | 확인 | SKIP",
  "issues": ["확인 필요 사항 목록"],
  "ok": ["적정 근거 목록"],
  "evidence": ["첨부파일 확인 내용"]
}
```

**status 기준**:
- `적정`: 모든 체크리스트 충족, 금액 한도 이내, 증빙 완비
- `확인`: 하나 이상 미충족 또는 불명확 사항 있음
- `SKIP`: 전자세금계산서 자동 연동 등 시스템 내장 증빙으로 별도 검토 불필요한 경우

## IV. 자주 발생하는 주의사항 (캠퍼스타운 특이사항)

1. **전담조직 소속 인력**: 회의비, 자문비, 심사평가비 지급 불가 → 지급대상 확인 필수
2. **운영비성 경비**: 대학 정규 전담인력에게 지급 불가
3. **공간임차료**: 시비(보조금)로 집행 불가 (대응자금만 가능)
4. **인테리어**: 집행 불가 (2025.7 명확화)
5. **동일업체 수의계약**: 연 4회 초과 금지 (2025.7 신설)
6. **시외여비**: 서울시 및 자치구 사전 승인 문서 필수 (2025.7 신설)
7. **소상공인 간편결제**: 서울시 사전 협의 후 사용 가능
8. **물품구입 원칙**: 임차(리스) 우선, 구입 시 세부실행계획서 승인 필요
9. **창업지원금 불가항목**: 인건비, 자산취득비
10. **일괄용역 불가**: 프로그램용역비에서 사업 전체를 외부 용역으로 위탁 불가
11. **창업지원금+창업시상금 중복**: 동일 팀에게 양자 동시 지급 허용 (2024.3 개정)
12. **대응자금 우선 집행**: 보조금 집행 전 대응자금 먼저 집행 원칙
13. **글로벌 프로그램 해외 전시**: 캠퍼스타운 기업성장센터 주관 CES 등 해외 전시에 시비 투입 불가, 항공료·해외체류비는 기업 자부담 [근거: 붙임4]
14. **부가가치세 환급금**: 집행금액에서 공제 또는 반납 원칙 (재투자 시 서울시 사전 승인)
