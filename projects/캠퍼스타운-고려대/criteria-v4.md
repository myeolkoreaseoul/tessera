## I. 증빙 평가 규칙 (★ v4 신설)

> 판정 시 evidence.json + 실제 다운로드 파일을 기준으로 증빙 충족 여부를 판단한다.

### 1. 보탬e 집행방식별 시스템 증빙 [근거: 지침 p.11 집행등록 방법]

| 집행방식(evidenceType) | 자동 충족 증빙 | 근거 |
|---|---|---|
| 전자세금계산서 | 세금계산서 | p.11: "전자(세금)계산서 발행정보를 조회 후 집행등록" |
| 보조금전용신용카드 | 카드영수증 | 제16조②, p.11: "카드매입내역을 조회하여 집행등록" |
| 기타 | 없음 | 첨부파일로 개별 확인 |

→ 전자세금계산서/보조금카드 건은 **결제 증빙(영수증/계산서)을 별도 첨부하지 않아도 적정**

### 2. 첨부파일 존재 = 증빙 첨부됨 [근거: 지침 p.11 파일첨부 대상]

- evidence.json `files` 배열에 파일명이 있고, 실제 다운로드 확인되면 → **해당 증빙이 보탬e에 첨부된 것**
- 파일명으로 증빙 종류를 추론하여, 비목별 필수 증빙과 매칭
- 파일이 비어있거나 다운로드 실패 시 → "확인"

### 3. 공과금 증빙 인정 범위 [근거: p.26 증빙서류 표, p.11 파일첨부 대상]

지침 p.26 공간운영비 증빙: **(공과금)영수증, 카드영수증(또는 세금계산서, 계좌이체확인증)**
지침 p.11 파일첨부 대상: **공공요금 자동이체영수증, 지로·고지서납부영수증**

따라서 다음은 유효한 공과금 증빙:
- 납부확인서, 납부영수증
- 자동이체영수증, 지로납부영수증, 고지서납부영수증

**이메일청구서의 증빙 인정**: 이메일청구서 파일 자체는 청구 내역을 확인하는 서류이며, **납부 사실의 증빙**은 다음 중 하나로 충족:
1. 증빙유형이 **전자세금계산서** 또는 **보조금카드** → 시스템 결제정보로 납부 확인 (추가 증빙 불필요) [I.1]
2. 증빙유형이 **기타**인 경우 → 이메일청구서 + **계좌이체확인증/카드영수증** 등 결제증빙이 함께 첨부되어야 유효
3. 이메일청구서 PDF 내에 **납부확인**, **이체완료**, **수납완료** 등 납부 사실이 기재된 경우 → 유효

**공과금은 비교견적서 불필요** [근거: p.26 증빙서류 표 — 공간운영비 증빙 항목에 견적서(비교견적서) 미포함]

### 4. "확인" 판정 기준 — 객관적 위반사항만

**"확인"으로 판정하는 경우:**
- 금액 한도 초과 (비목별 기준 위반)
- 지원불가항목 해당 (인건비, 자산취득비 등)
- 재원 위반 (시비 투입 불가 비목에 시비 투입)
- 증빙파일 미존재 (files 빈 배열 또는 다운로드 실패)
- 회계연도 초과 (집행일 > 2025-12-31) [제16조⑤]
- 인테리어 비용 [제19조]
- 2천만원 초과 계약에 나라장터 미이용 확인 필요 [제17조⑦]

**"확인"으로 판정하지 않는 경우** (데이터에서 확인 불가능):
- 서명 여부 (파일 내부 확인 불가)
- 전담조직 소속 여부 (소속 정보 없음)
- 정규직/기간제 구분 (고용형태 정보 없음)
- 이력서 최초 1회 첨부 여부 (이전 집행 이력 없음)
- 사전 승인 문서 존재 여부 (첨부파일 내용 미확인 시)

→ 위 사항들은 **파일이 존재하면 ok에 기록**, 파일 자체가 없을 때만 issues

## II. 공통 규정

### 1. 집행 원칙 [근거: 지침 제4조, 제16~17조]
- 인건비성 경비 제외한 모든 비목: **보조금결제 전용카드** 사용 원칙 [제16조②]
- 지출거래 시 **전자세금계산서** 사용 원칙 [제16조②]
- 소상공인 간편결제 사용 시 **서울시 사전 협의** 필요 [제16조② 단서]
- 현금 인출 불가; 부득이한 경우 서울시 사전 승인 [제16조④]
- 지출원인행위 없이 임의 인출 금지, "일괄 인출 후 사후정산" 불가 [제16조③]
- 사업비는 회계연도 말(12.31)까지 집행, 이월 불가 (사전승인 시 이월 가능, 이월신청서 요건 충족) [제16조⑤]

### 2. 계약 규정 [근거: 지침 제17조]
- 추정가격 **2천만원 초과**: 나라장터 이용 필수 [제17조⑦]
- 추정가격 **2천만원 이하**: 나라장터 이용 **의무** (수의계약 포함) [제17조⑦]
- 수의계약 **동일업체 연 4회 초과 금지** (2025.7 신설) [제17조⑦]
- 추정가격 1천만원 이하 물품 임차는 일반지출 가능 [제17조⑦]

### 3. 사업비 집행 우선순위 [근거: 지침 제17조①]
- 대학 대응자금을 보조금(시비)보다 **우선 집행** [제17조①]

### 4. 사업계획 변경 [근거: 지침 제17조②]
- 연 **4회**까지 (11월 30일까지 서울시 사전 승인)
- AI 인재양성 프로그램 추가 시 **연 5회**까지 [제17조②]

### 5. 인건비성 경비 원천징수 [근거: 지침 제18조②]
- 강사비, 회의비, 자문비, 심사평가비, 멘토비, 단순인건비 등
- 기타소득이 **건당 125,000원 초과** 시 원천징수 의무
- 원천징수영수증 첨부 필수
- ※ 지침 제18조②는 "건당", 붙임6 하단 주석은 "월"로 표현이 병존하나, 소득세법 제84조(기타소득 과세최저한)에 따라 **건별 기준** 적용

### 6. 100만원 이상 거래 시 비교견적서 [근거: 지침 붙임6]
- **100만원 이상** 물품/용역 거래 시 견적서 + 비교견적서 첨부
- **적용 제외**: 공과금(p.26 증빙표에 미포함), 인건비성 경비(붙임6에서 제외 명시)
- ※ 단기임차료·물품임차비는 p.25-26 증빙표에 "견적서(비교견적서)"가 명시되어 있으므로 면제 대상 아님
- ※ 공간임차료는 p.26 증빙표에 견적서 대신 "주변 시세 현황"이 요구되므로 비교견적서 불필요 (별도 취급)

### 7. 물품 관리 [근거: 지침 제19조]
- 물품·장비 확보는 **임차(리스) 원칙** [제19조①]
- 단가 50만원 이상 물품은 중요재산 관리 대상 [제19조④]
- **인테리어 비용 집행 불가** [제19조, 2025.7 개정]
- **임차시설 원상복구 비용**: 임차기간 만료 후 발생하는 원상복구 비용은 사업예산으로 집행 불가 [제19조]

### 8. 부가가치세 환급금 [근거: 지침 제22조(제24조⑥)]
- 부가가치세 환급금은 **정산 시 집행금액에서 공제 또는 반납**하는 것이 원칙
- 환급금을 사업에 재투자하고자 할 경우 **서울시 사전 승인** 필요
- ※ 건별 집행 검토가 아닌 총괄 정산 수준에서 확인하는 항목 (인적 검토 시 확인)

### 9. 정산 불인정 기준 [근거: 지침 제23조②]
다음에 해당하는 집행은 정산 시 불인정:
1. **교부조건 또는 지침 위반** 집행
2. **보조사업과 무관한 지출**
3. **사업계획 변경·이월 무승인** 집행

## III. 비목별 집행기준 및 증빙서류

### [프로그램추진경비]

#### 2-1. 강사비 [e나라도움: 기타인건비(110-05)]

**집행기준** [근거: II.집행기준 p.19]:
- 서울시 인재개발원 강사료 및 원고료 지급기준 준용
- 비대면: 실시간 화상교육만 인정 (녹화 강의 불인정)

**필수 증빙서류** [근거: p.25]:
- 내부결재(계획·결과보고)
- 이력서(최초1회)
- 강의확인서(서명)
- 강의증빙자료(교안, 강의사진, 출석부 중 택1)
- 계좌이체확인증
- 원천징수영수증(해당 시)

**적정 조건**:
- 첨부파일에 강의 관련 문서가 1개 이상 존재
- 집행방식이 계좌이체/카드 등 정상 결제
- 금액이 인재개발원 기준 범위 내 (명백한 초과 아닌 한)

**확인 사유**:
- 첨부파일 없음 (files 빈 배열)
- 건당 125,000원 초과인데 원천징수 관련 파일 없음 [제18조②]
- 비대면 강의로 추정되나 실시간 여부 불명확 (목적에 "비대면"/"온라인" 포함 시)

#### 2-2. 심사 및 평가비 [e나라도움: 기타인건비(110-05)]

**집행기준** [근거: II.집행기준 p.19]:
- 시간당 **10만원** 이내, 1일 최대 **50만원** 이내
- 서면 심사·평가: **7만원** 이내 (1일)
- 캠퍼스타운 **전담조직 소속 인력 지급 불가**

**필수 증빙서류** [근거: p.25]:
- 내부결재(계획·결과보고)
- 참석자명부(서명)
- 심사 또는 평가결과서
- 이력서(최초1회)
- 계좌이체확인증
- 원천징수영수증(해당 시)

**적정 조건**:
- 첨부파일에 심사/평가 관련 문서 존재
- 지방비 금액 ≤ 500,000원 (1일 한도)
- 집행방식 정상

**확인 사유**:
- 첨부파일 없음
- 지방비 금액 > 500,000원 → 1일 50만원 한도 초과 가능성
- 건당 125,000원 초과인데 원천징수 관련 파일 없음

#### 2-3. 멘토비 [e나라도움: 기타인건비(110-05)]

**집행기준** [근거: II.집행기준 p.19]:
- 대면: 1일 **30만원** 이내
- 비대면(원격회의): 1일 **15만원** 이내
- **1인 멘토 월 200만원 초과 지급 불가**

**필수 증빙서류** [근거: p.25]:
- 멘토링내역서(서명)
- 지급내역서
- 계좌이체확인증
- 이력서·경력증명서(최초1회)
- 원천징수영수증(해당 시)

**적정 조건**:
- 첨부파일에 멘토링 관련 문서 존재
- 지방비 금액 ≤ 300,000원 (대면 1일 한도)
- 집행방식 정상

**확인 사유**:
- 첨부파일 없음
- 지방비 금액 > 300,000원 → 1일 30만원 한도 초과 가능성
- 건당 125,000원 초과인데 원천징수 관련 파일 없음
- 동일 멘토 월 누적 확인 필요 (동일인 반복 시)

#### 2-4. 회의비 및 자문비 [e나라도움: 기타인건비(110-05)]

**집행기준** [근거: II.집행기준 p.19]:
- 대면: 2시간 이내 **15만원**, 초과 시 **20만원** (1일)
- 비대면: 2시간 이내 **5만원**, 초과 시 **7만원** (1일)
- **전담조직 소속 인력 지급 불가**

**필수 증빙서류** [근거: p.25]:
- 회의참석확인서(서명)
- 회의록
- 이력서(최초1회)
- 계좌이체확인증
- 원천징수영수증(해당 시)

**적정 조건**:
- 첨부파일에 회의 관련 문서 존재
- 지방비 금액 ≤ 200,000원 (대면 한도)
- 집행방식 정상

**확인 사유**:

## IV. results.json 작성 형식

```json
{
  "rowNum": 1,
  "type": "비목명",
  "purpose": "집행용도",
  "amount": 금액,
  "vendor": "거래처",
  "date": "집행일",
  "status": "적정 | 확인 | SKIP",
  "issues": ["확인 필요 사항"],
  "ok": ["적정 근거"],
  "evidence": ["첨부파일 확인 내용"],
  "opinion": "검토의견 (보탬e 입력용)",
  "files_found": ["확인된 파일명"],
  "files_missing": ["누락 증빙"]
}
```

**status 기준**:
- `적정`: 필수 증빙 파일 존재 + 금액 한도 이내 + 명백한 규정 위반 없음
- `확인`: 증빙 파일 미존재, 금액 한도 초과, 지원불가항목, 재원 위반 등
- `SKIP`: 전자세금계산서 시스템 연동 등 별도 검토 불필요

## V. 자주 발생하는 주의사항

1. **전담조직 소속 인력**: 회의비, 자문비, 심사평가비 지급 불가 → 의심 근거 있을 때만
2. **운영비성 경비**: 대학 정규 전담인력에게 지급 불가
3. **공간임차료**: 시비 투입 불가 → 지방비 > 0이면 무조건 확인
4. **인테리어**: 집행 불가 [제19조]
5. **동일업체 수의계약**: 연 4회 초과 금지 [제17조⑦]
6. **시외여비**: 사전 승인 문서 필수 [p.26]
7. **창업지원금 불가항목**: 인건비, 자산취득비 (가구, 사무용복합기, 태블릿PC, 소프트웨어, 바이러스 백신 등)
8. **일괄용역 불가**
9. **공과금 증빙**: 납부확인서, 납부영수증, 자동이체영수증 등이 유효. **이메일청구서 단독은 불인정** (결제증빙과 함께 있어야 유효)
10. **비교견적서 100만원**: 물품/용역 거래에 적용, 공과금·인건비 제외. 단기임차료·물품임차비는 면제 아님 (공간임차료는 별도 — 주변 시세 현황으로 대체)
11. **원천징수**: 건당 125,000원 초과 시에만 [제18조②]
12. **부가가치세 환급금**: 정산 시 공제/반납 원칙, 재투자 시 서울시 사전 승인 [제22조] (총괄 정산 수준 확인)
13. **정산 불인정**: 교부조건/지침 위반, 보조사업 무관 지출, 변경·이월 무승인 집행 [제23조②]
14. **임차시설 원상복구**: 임차기간 만료 후 원상복구 비용은 사업예산으로 집행 불가 [제19조]
